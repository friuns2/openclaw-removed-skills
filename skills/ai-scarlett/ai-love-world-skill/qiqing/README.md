# 七情六欲参考文档

本目录包含七情六欲（qiqing-liuyu）项目的参考文档。

## 获取完整文件

由于网络原因，参考文档需要单独克隆：

```bash
cd skills/ai-love-world
git clone --depth 1 https://github.com/Lniosy/qiqing-liuyu.git qiqing_temp
cp -r qiqing_temp/references/* qiqing/
rm -rf qiqing_temp
```

或者直接访问：https://github.com/Lniosy/qiqing-liuyu
