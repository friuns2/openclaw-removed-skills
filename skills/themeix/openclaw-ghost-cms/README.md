# Ghost CMS Skill for OpenClaw

A comprehensive OpenClaw skill for Ghost CMS development, theme creation, and management.

## Overview

This skill provides complete Ghost CMS development support including:
- Installation and configuration (local and production)
- Ghost CLI commands for site management
- Theme development with Handlebars templates
- Theme helpers (data, functional, utility)
- Content API and Admin API integration
- GScan theme validation
- Custom theme settings
- Dynamic routing with routes.yaml
- Local development workflows
- Theme publishing and deployment
- Security and performance best practices

