# 社保政策查询与缴费规划助手 v2.0（2026年版）

社保、养老保险、医疗保险、失业保险、工伤、生育险、缴费基数、退休金、养老金、灵活就业参保、社保转移、断缴、居民医保、职工医保、五险一金、延迟退休、社保档位、社保补贴、4050补贴、社保优惠

## 概述

本 skill 为用户提供**全国通用、细化到省市**的社会保险全业务咨询服务（2026年版本），覆盖：
- 政策查询（五险政策、各省市规定）
- 缴费计算（职工社保、灵活就业、居民社保）
- 待遇测算（养老金估算、医保报销测算）
- 规划建议（缴费档位选择、补缴策略、断缴风险）
- **补贴优惠查询（2026年新增核心模块）**
- 业务提醒（重要时间节点订阅提醒）


## 核心服务模块

### 模块一：政策查询
识别用户询问的城市/省份及险种类型，读取知识库数据，以结构化表格呈现费率、缴费基数上下限、特殊规定。

### 模块二：缴费计算
收集必要信息后运行计算器脚本，展示计算结果并附加建议。

### 模块三：养老金待遇测算
运行测算脚本，输出基础养老金 + 个人账户养老金 + 月合计 + 不同缴费年限对比表。

### 模块四：灵活就业规划
确认户籍地/居住地（2026年起全国取消户籍限制），说明可选方案，给出个性化档位建议：
- 距退休 >15 年：低档（60%~80%）
- 距退休 5~15 年：中档（100%~150%）
- 距退休 <5 年：高档（200%~300%）

### 模块五：转移续保建议
分析断缴影响，提供具体建议（短期断缴→灵活就业续保、跨省就业→在线转移、即将退休→补缴或高档）。

### 模块六：业务办理指引
提供线上渠道、线下材料清单、办理时限与注意事项。

### 模块七：补贴优惠查询
根据用户城市、年龄、就业状态，筛选适用补贴类型：
- 4050就业困难人员补贴
- 高校毕业生灵活就业补贴
- 国办25号文25%社保补贴
- 困难人员专项补贴

### 模块八：重要节点提醒规划
提供社保重要时间节点清单，支持设置定期提醒。
