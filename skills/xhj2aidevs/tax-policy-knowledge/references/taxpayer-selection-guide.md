# 纳税人身份选择指南

## 一、一般纳税人 vs 小规模纳税人 核心区别

| 对比维度 | 一般纳税人 | 小规模纳税人 |
|---------|-----------|-------------|
| 认定标准 | 年应税销售额 > 500万 | 年应税销售额 ≤ 500万 |
| 增值税税率 | 13%/9%/6%/0% | 3%（2027年底前减按1%） |
| 进项抵扣 | ✅ 可抵扣进项税额 | ❌ 不可抵扣 |
| 发票类型 | 增值税专用发票 | 增值税普通发票（可申请专票代开） |
| 申报周期 | 按月申报 | 按季申报 |
| 会计核算 | 要求健全的会计核算 | 要求较宽松 |

## 二、税负平衡点决策法

### 核心公式
当增值率（毛利率）达到 **28.8%** 时，两种纳税人身份税负相同。

**增值率 = (销售额 - 进项成本) / 销售额 × 100%**

### 决策规则
| 你的增值率 | 推荐身份 | 原因 |
|-----------|---------|------|
| **> 28.8%** | 小规模纳税人 | 税负更低 |
| **< 28.8%** | 一般纳税人 | 进项抵扣优势大 |
| **= 28.8%** | 两者税负相同 | 考虑客户开票需求 |

### 示例
- 年销售额 400万，进货成本 300万
- 增值率 = (400-300)/400 = 25%
- **结论**：25% < 28.8%，选择一般纳税人更优

## 三、快速决策流程

```
年销售额 > 500万?
├─ 是 → 强制登记为一般纳税人
└─ 否 → 增值率 > 28.8%?
      ├─ 是 → 小规模纳税人（税负低）
      └─ 否 → 一般纳税人（抵扣优势）
```

## 四、特殊情况考虑

1. **客户要求**：若大客户要求专票抵扣，优先考虑一般纳税人
2. **行业特性**：商贸业增值率低，适合一般纳税人；服务业增值率高，适合小规模
3. **政策窗口**：2027年底前小规模减按1%征收率，阶段性更优惠
