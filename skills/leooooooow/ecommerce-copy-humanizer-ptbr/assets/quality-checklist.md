# Checklist de qualidade — PT-BR humanizado

Passe por esta lista antes de entregar qualquer copy.

## 1. Voz e pessoa (5)
- [ ] Trata o leitor por "você" consistentemente
- [ ] Usa verbos ativos, não passiva
- [ ] Zero ocorrência de "nossa empresa" / "nós acreditamos"
- [ ] Nenhuma frase começa com "Apresentamos/Conheça/Descubra/Experimente"
- [ ] Nenhum gerúndio infinito ("estaremos enviando")

## 2. Concretude (6)
- [ ] Pelo menos 1 número por parágrafo (medida, tempo, preço, quantidade)
- [ ] Pelo menos 1 cena de uso específica (onde, quando)
- [ ] Pelo menos 1 material/ingrediente/especificação real
- [ ] Nenhum fato inventado — tudo veio do usuário ou do produto
- [ ] Objeção principal do produto respondida
- [ ] Prazo, frete, garantia ou pagamento mencionados quando relevante

## 3. Blocklist zero (5)
- [ ] Nenhuma ocorrência de "solução", "versátil", "inovador"
- [ ] Nenhuma ocorrência de "premium", "exclusivo", "único" sem prova
- [ ] Máximo 1 advérbio em "-mente" por parágrafo
- [ ] Nenhum bullet paralelo começando com o mesmo verbo
- [ ] Nenhuma tradução literal do inglês ("mudança de jogo" etc.)

## 4. Ritmo (5)
- [ ] Alternância entre frases curtas e médias
- [ ] Nenhuma frase com mais de 25 palavras sem pontuação forte
- [ ] Nenhuma sequência de três adjetivos
- [ ] Nenhum parágrafo com mais de 5 linhas visuais
- [ ] Lê bem em voz alta sem travar

## 5. CTA (4)
- [ ] CTA tem verbo de ação específico (não "clique aqui")
- [ ] CTA oferece motivo para agir agora
- [ ] Urgência, se usada, é real e verificável
- [ ] Duas variações (A/B) fornecidas

## 6. Regionalização (5)
- [ ] Zero portuguesismos ("fato" = terno, "sandes", "autocarro")
- [ ] Referências culturais são brasileiras
- [ ] Pagamento menciona pix/cartão/boleto quando faz sentido
- [ ] Medidas em padrão brasileiro (cm, kg, R$)
- [ ] Tamanhos em padrão brasileiro (P/M/G não S/M/L) quando roupa

## 7. Estrutura da entrega (5)
- [ ] Diagnóstico antes/depois presente
- [ ] Justificativa de cada mudança listada
- [ ] Duas variações A/B de título e CTA
- [ ] Avisos sobre o que precisa ser confirmado pelo usuário
- [ ] Output segue o template em `references/output-template.md`

## 8. Revisão final (5)
- [ ] Leu em voz alta duas vezes
- [ ] Nenhum erro de português
- [ ] Nenhuma inconsistência factual interna
- [ ] Tom de voz constante do início ao fim
- [ ] Se fosse recomendação de amigo, faria sentido? Sim/Não

**Mínimo aceitável: 35/40 itens marcados. Abaixo disso, reescrever.**
