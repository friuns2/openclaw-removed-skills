# Continuity Ledgers

## Goal

Prevent long-form collapse by turning volatile continuity into canonical records.

## Required ledger set

### 世界规则账本
Use `05B_世界规则账本.md` for:
- world laws
- system limits
- faction rules
- rule exceptions and their costs

### 伏笔回收台账
Use `05C_伏笔回收台账.md` for:
- planted clues
- unresolved mysteries
- expected payoff windows
- recovered / expired / transformed statuses

### 关系状态表
Use `05D_关系状态表.md` for:
- current alliances and fractures
- trust level shifts
- emotional stage changes
- relationship promises and unresolved tension

### 能力与资源变化表
Use `05E_能力与资源变化表.md` for:
- power growth
- item ownership
- authority changes
- resource gains and losses

## Update rule

These ledgers are parent-owned canonical files.

Child subagents may suggest updates, but the parent updates the ledger set only after downstream review passes.
