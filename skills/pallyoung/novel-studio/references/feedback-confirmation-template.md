# Feedback Confirmation Template

Use this pattern when the agent detects likely formal revision feedback from normal conversation.

---

## Suggested response pattern

我检测到你这句话**可能是一条正式反馈**。

### 我理解到的反馈
- 反馈内容：...
- 反馈类型：风格 / 剧情 / 人物 / 结构 / 方向

### 我判断的影响范围
- 可能影响的阶段：...
- 可能影响的文件：...

### 与旧设定的关系
- 是否与旧设定冲突：是 / 否 / 不确定
- 如果冲突，冲突点：...

### 需要你确认的事
1. 要不要把这条作为**正式反馈**记录？
2. 这是要**覆盖旧设定**，还是**在旧设定上新增要求**？
3. 是否按我判断的影响范围处理？

如你确认，我再进入修订模式并更新相关产物。
