# Anti-Template Checklist

## Goal

Catch formula drift before it becomes the book's default operating mode.

## Red flags

- 只有题材名，没有真正的推进发动机
- 只有“设定很酷”，没有代价和总主线
- 开篇只会炸一下，十章后主线发虚
- 主角像功能插件，不像有欲望和代价的人
- 支线只是挂件，不改变选择和后果
- 替换角色名后仍成立
- 场景换角色名后仍成立
- 章节总结只能写成“推进剧情 / 角色互动 / 埋伏笔”

## When to run it

Run this checklist at least three times:
- during lane selection
- during the opening gate
- during proofreading

When the Cliche Exhaustion Loop is active:
- `quick` mode in Discovery should surface the obvious default version fast
- `deep` mode in Story Planning should test whether the retained direction is only “a slightly decorated cliché”

## Chapter-level check

Ask:
- what changes in consequence, emotion, relationship, knowledge, or cost?
- what makes this chapter impossible to swap into another generic book?
- if I replace the character names, does the scene still read the same?

If yes, the chapter is drifting into template mode.
