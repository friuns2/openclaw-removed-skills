# meta-cognition-assistant【自研系统技能】

**版本**：v0.1.0  
**定位**：元认知进化系统助手 - 手动触发知识搜集、分析和整合

---

## 📖 技能说明

这是「元认知进化系统」的手动调用接口，让你可以：
- 立即触发知识搜集（不等待每日 Cron）
- 手动执行深度分析
- 查询知识库状态
- 触发技能整合

**对应的系统**：元认知进化系统（Cron 任务自动运行）

| 场景 | 示例 |
|------|------|
| 立即搜集信息 | 「立即搜集今天的 GitHub Trending」 |

## 🚀 使用方法

### 1. 立即搜集信息

```
立即搜集 [信息类型] 的信息

示例：
- 立即搜集今天的 GitHub Trending
- 立即搜集 AI 领域的最新进展
- 立即搜集老钱家族的决策案例
```

### 2. 手动深度分析

```
深度分析 [主题]

示例：
- 深度分析 DeerFlow 的技能设计
- 深度分析桥水基金的原则
- 深度分析 Elon Musk 的第一性原理
```

### 3. 查询知识库

```
查询知识库 [查询内容]

示例：
- 查询知识库中有多少专家模式
- 查询知识库中关于决策框架的内容
- 查询本周搜集了多少信息
```

### 4. 触发整合

```
整合 [时间范围] 的 [内容类型]

示例：
- 整合本月的技能升级
- 整合本周的洞察
- 整合今日的搜集结果
```

## 🔗 与 Cron 任务的关系

**Cron 任务**（自动运行）：
- 每日信息搜集（08:00）
- 每周深度分析（周日 20:00）
- 月度技能整合（月末 22:00）

**本技能**（手动调用）：
- 立即触发上述任务
- 查询执行结果
- 自定义分析主题

**建议使用**：
- 日常：依赖 Cron 任务自动运行
- 特殊需求：使用本技能手动触发

## 💡 最佳实践

1. **不要频繁手动触发**
   - Cron 任务会自动执行
   - 手动触发用于特殊需求

2. **提供具体主题**
   - 「分析 DeerFlow」比「分析竞品」更好
   - 具体的主题 = 更好的结果

3. **查看执行日志**
   - 执行后查看 `memory/evolution/daily/`
   - 了解搜集/分析结果
