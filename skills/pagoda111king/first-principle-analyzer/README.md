## 📖 技能说明

First Principle Analyzer 是一款第一性原理思维增强工具，通过结构化 7 阶段分析框架（问题接收→假设识别→逐层分解→基本真理验证→重构方案→类比方案对比→报告生成），帮助用户从基本原理出发分析问题，识别并挑战隐含假设，提取不可再分的基本真理，从 fundamentals 演绎创新解决方案。

**核心价值**:
- 摆脱类比思维（「别人怎么做」）
- 回归基本真理（「物理上可能的是什么」）
- 重构创新方案（从基本真理演绎）

## 📋 分析流程

```
阶段 1: 问题接收与初步分析 → 问题分类
    ↓
阶段 2: 假设识别与质疑 → 至少 3 个隐含假设
    ↓
阶段 3: 逐层分解（5 Why）→ 基本真理
    ↓
阶段 4: 基本真理验证 → 验证通过
    ↓
阶段 5: 重构方案生成 → 至少 2 个创新方案
    ↓
阶段 6: 类比方案对比 → 创新 vs 传统
    ↓
阶段 7: 报告生成 → 完整 Markdown 报告
```

## 📚 实战案例

### 案例 1：SpaceX 火箭成本分析

**问题**: 如何降低火箭发射成本？

**分析过程**:
- 假设识别：「火箭必须是一次性的」
- 5Why 分解：为什么火箭必须一次性？→ 技术限制 → 可回收技术可行
- 基本真理：材料成本仅占售价 2%
- 重构方案：可回收火箭

**结果**: 材料成本仅占售价 2%，差距来自制造方式和供应链

---

### 案例 2：职业规划分析

**问题**: 我应该换工作还是创业？

**分析过程**:
- 假设识别：「创业必须辞职全职投入」
- 5Why 分解：为什么必须辞职？→ 传统观念，非必然
- 基本真理：创业的本质是验证价值假设
- 重构方案：副业验证，周末 MVP

**结果**: 职业价值 = 能力×稀缺性×杠杆，而非 title 或薪资

---

### 案例 3：技能系统设计

**问题**: 如何创建一个更好的技能系统？

**分析过程**:
- 假设识别：「技能系统必须有中央市场」
- 5Why 分解：为什么必须有中央市场？→ 历史沿袭，非技术必然
- 基本真理：技能的本质是可复用的能力单元
- 重构方案：基于能力图谱的自动发现系统

**结果**: 从「人找技能」变为「技能找人」

## ❓ FAQ

### Q1: 什么是第一性原理？

**A**: 第一性原理是从基本原理出发推理问题的思维方式，而不是通过类比或经验。它要求识别并挑战隐含假设，提取不可再分的基本真理，然后从这些真理演绎创新方案。

### Q2: 什么时候使用这个技能？

**A**: 当面对复杂问题、重大决策、或传统方案不满意时。特别是需要深度思考、创新方案、或挑战现有假设的场景。

### Q3: 分析需要多长时间？

**A**: 完整 7 阶段分析通常需要 30-60 分钟。简单问题可能更快，复杂问题可能需要更长时间。

### Q4: 如何验证分析结果？

**A**: 分析结果应该：
- 基于可验证的基本真理
- 逻辑自洽
- 有实际可行性
- 与传统方案有明显差异

### Q5: 可以分析任何问题吗？

**A**: 理论上可以，但最适合：
- 复杂问题（多因素、多约束）
- 重大决策（高影响、难逆转）
- 创新需求（需要突破传统思维）

### Q6: 如何保存分析结果？

**A**: 分析结果会以 Markdown 格式输出，可以：
- 保存到文件
- 分享给团队
- 作为决策依据
- 定期回顾更新
