# AI商拍工具完整参考文档

本文档是MCP Server提供的AI商拍模块工具集合,包含以下工具的完整API说明:

- **模特试穿试戴**（`model_try_clothes`）：让AI模特试穿指定衣物，生成试穿效果图
- **商品场景图**（`business_product_scene`）：将商品放入指定场景，生成场景化展示效果图
- **混云图片生成**（`hy_image`）：根据参考图和提示词生成图片，支持自定义尺寸

## 如何选择工具

| 用户需求                  | 使用工具                     |
| --------------------- | ------------------------ |
| 让模特穿上衣服/裤子/套装         | `model_try_clothes`      |
| 把商品（家电/家具/美妆等）放入场景中展示 | `business_product_scene` |
| 根据参考图和提示词生成自定义图片      | `hy_image`               |

## 模特试穿试戴

### model\_try\_clothes

**功能说明**：让模特试穿指定的衣服、裤子、衣服+裤子（套装）。

**使用场景**：

- 让模特穿上指定的衣服、裤子、套装
- 在调用时需要上传衣服、裤子、套装的图片

**请求参数**：

| 参数         | 类型     | 必填     | 默认值    | 说明                                            |
| ---------- | ------ | ------ | ------ | --------------------------------------------- |
| `url`      | string | 是      | -      | 公网URL或本地**完整路径**                              |
| `gender`   | string | 是      | -      | 模特性别，可选值：`男`、`女`                              |
| `content`  | string | 否      | 随机     | 服装类型，非必填，上装、裤装、套装、裙子、外套等                      |
| `category` | string | 否      | 室内商拍   | 拍摄场景分类，可选值：`室内商拍`、`稚趣童装`、`女生街拍`、`户外行摄`、`男士街拍` |
| `feature`  | object | 否      | -      | 人物特征配置，详见下方 [feature 字段说明](#feature-字段说明)     |
| <br />     | <br /> | <br /> | <br /> | <br />                                        |
| `prompt`   | string | 否      | -      | 描述画面背景和氛围，例如：中远景，天空有小鸟，阳光明媚的公园，背景有咖啡厅等        |

#### feature 字段说明

所有字段均为非必填，值直接传入中文描述：

| 字段           | 类型     | 说明 | 默认值 | 示例值         |
| ------------ | ------ | -- | :-- | ----------- |
| `age`        | string | 年龄 | 青年  | 儿童、青年、中年、老年 |
| `hair_style` | string | 发型 | -   | 短发、长发、卷发、寸头 |
| `hair_color` | string | 发色 | -   | 黑色、金色、棕色、白色 |
| `color`      | string | 肤色 | -   | 白皙、自然、小麦色   |
| `figure`     | string | 身材 | -   | 苗条、健壮、匀称    |
| `expression` | string | 表情 | -   | 微笑、严肃、自然    |

**调用示例**：

示例1：基础调用（仅传入衣服图片）

```json
{
  "url": "https://example.com/clothes.jpg"
}
```

示例2：指定性别和场景分类

```json
{
  "url": "https://example.com/clothes.jpg",
  "gender": "女",
  "category": "女生街拍",
  "feature":{
    "age":"青年"
  }
}
```

示例3：完整参数（含人物特征配置）

```json
{
  "url": "https://example.com/clothes.jpg",
  "gender": "女",
  "category": "室内商拍",
  "feature": {
    "age": "青年",
    "hair_style": "长发",
    "hair_color": "黑色",
    "color": "白皙",
    "figure": "苗条",
    "expression": "微笑"
  }
}
```

**返回值说明**：

成功返回：

```json
{
  "code": 0,
  "msg": "ok",
  "data": {
    "_id": "调用链追踪ID"
  }
}
```

失败返回：

```json
{
  "code": 1001,
  "msg": "图片URL无效",
  "data": null
}
```

> **注意**：
>
> - 返回结果中 `code!=0` 时，表示调用失败，需要检查参数是否正确
> - 任务提交成功后需要使用`get_job_status`查询生成进度和结果

## 商品场景图

### business\_product\_scene

**功能说明**：将商品图片放入指定场景生成场景化效果图。支持家电、家具、美妆等商品在不同场景展示。

**使用场景**：

- 在家电、家具商品图片生成场景化展示效果图
- 在美妆商品图片生成手持或户外场景展示效果图

**请求参数**：

| 参数       | 类型     | 必填 | 默认值 | 说明                                                      |
| -------- | ------ | -- | --- | ------------------------------------------------------- |
| `url`    | string | 是  | -   | 商品图片公网URL或本地**完整路径**                                    |
| `type`   | string | 否  | -   | 商品类型，可选值：`家电`、`家具`、`美妆`                                 |
| `scene`  | string | 否  | -   | 商品场景，需与 `type` 匹配，详见下方 [scene 参数映射](#scene-参数映射)        |
| `ratio`  | string | 否  | 3:4 | 生成图比例，可选值：`21:9`、`16:9`、`4:3`、`1:1`、`3:4`、`9:16`、`9:21` |
| `prompt` | string | 否  | -   | 指定场景、商品类型、生成比例等                                         |

#### scene 参数映射

不同商品类型支持的场景选项不同：

| 商品类型 (`type`) | 支持的场景 (`scene`) |
| ------------- | --------------- |
| 家电            | `厨房`、`阳台`、`客厅`  |
| 家具            | `厨房`、`阳台`、`客厅`  |
| 美妆            | `手持`、`户外`       |

**调用示例**：

示例1：家电商品放入厨房场景

```json
{
  "url": "https://example.com/refrigerator.jpg",
  "type": "家电",
  "scene": "厨房",
  "ratio": "3:4",
  "prompt": "商品放入厨房场景，生成3:4比例的效果图"
}
```

示例2：美妆商品手持展示

```json
{
  "url": "https://example.com/lipstick.jpg",
  "type": "美妆",
  "scene": "手持",
  "ratio": "3:4",
  "prompt": "商品手持展示，生成3:4比例的效果图"
}
```

**返回值说明**：

成功返回：

```json
{
  "code": 0,
  "msg": "ok",
  "data": {
    "_id": "调用链追踪ID"
  }
}
```

失败返回：

```json
{
  "code": 1001,
  "msg": "图片URL无效",
  "data": null
}
```

> **注意**：
>
> - 返回结果中 `code!=0` 时，表示调用失败，需要检查参数是否正确
> - 任务提交成功后需要使用`get_job_status`查询生成进度和结果
> - `scene` 参数需与 `type` 匹配，否则可能调用失败

## 混云图片生成

### hy\_image

**功能说明**：混云图片生成工具。根据参考图和提示词生成图片，支持自定义尺寸。

**使用场景**：

- 根据一张或多张参考图生成新的图片
- 通过提示词控制生成图片的内容和风格
- 自定义生成图片的尺寸（宽、高均在\[512, 2048]像素范围内，宽高乘积≤1024\*1024像素）

**请求参数**：

| 参数           | 类型        | 必填 | 默认值       | 说明                                                          |
| ------------ | --------- | -- | --------- | ----------------------------------------------------------- |
| `images`     | string\[] | 否  | null      | 参考图URL列表，最多3张，可为null                                        |
| `prompt`     | string    | 是  | -         | 提示词，非必填，填了用参数里的                                             |
| `resolution` | string    | 否  | 1024:1024 | 生成图尺寸，宽、高均在\[512, 2048]像素范围内，宽高乘积≤1024\*1024像素，默认为1024:1024 |

**调用示例**：

示例1：仅使用提示词生成图片（无参考图）

```json
{
  "prompt": "一只可爱的小猫咪在花园里玩耍，阳光明媚"
}
```

示例2：使用单张参考图和提示词生成图片

```json
{
  "images": ["https://example.com/reference.jpg"],
  "prompt": "根据参考图生成类似风格的风景画"
}
```

示例3：使用多张参考图和自定义尺寸

```json
{
  "images": [
    "https://example.com/ref1.jpg",
    "https://example.com/ref2.jpg"
  ],
  "prompt": "融合参考图的风格生成一张商业产品摄影图",
  "resolution": "1920:1080"
}
```

示例4：仅使用参考图（无提示词）

```json
{
  "images": ["https://example.com/reference.jpg"],
  "resolution": "512:512"
}
```

示例5：完整参数调用

```json
{
  "images": [
    "https://example.com/ref1.jpg",
    "https://example.com/ref2.jpg",
    "https://example.com/ref3.jpg"
  ],
  "prompt": "生成一张具有参考图风格的现代科技感产品宣传图",
  "resolution": "1024:1024"
}
```

**返回值说明**：

成功返回：

```json
{
  "code": 0,
  "msg": "ok",
  "data": {
    "_id": "调用链追踪ID"
  }
}
```

失败返回：

```json
{
  "code": 1001,
  "msg": "图片URL无效",
  "data": null
}
```

> **注意**：
>
> - 返回结果中 `code!=0` 时，表示调用失败，需要检查参数是否正确
> - 任务提交成功后需要使用`get_job_status`查询生成进度和结果
> - `images` 参数最多支持3张参考图
> - `resolution` 参数的宽、高均需在\[512, 2048]像素范围内，且宽高乘积≤1024\*1024像素
