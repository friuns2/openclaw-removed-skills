# 插画

## 描述

扁平化插画风格，色块分明，几何化人物或物品，清新明快的配色。兼顾趣味性与信息传达。

## 适用场景

教程、科普、知识分享、生活方式类文章。适合轻松或专业调性的账号。

## Prompt 要点

- flat illustration style, vector-like shapes, solid color blocks, minimal gradients
- stylized characters or objects with geometric simplification
- bright and fresh color palette, 2-4 main colors
- clear visual hierarchy, one main scene or concept
- 画面内文字必须为中文，在 prompt 中直接写出要显示的中文文案
