# 大字报

## 描述

大胆醒目的排印设计，超大字号占据画面主体，强对比色块，海报感十足。适合观点鲜明、需要吸引眼球的文章。

## 适用场景

观点文章、评论、热点话题、犀利文风。适合个人 IP 或自媒体调性的账号。

## Prompt 要点

- bold typography-driven layout, oversized Chinese characters as main visual
- high contrast color blocks, 2-3 strong colors, no subtle gradients
- poster-like composition, graphic and punchy
- minimal imagery, text IS the visual; background supports readability
- 画面内文字必须为中文，在 prompt 中直接写出要显示的中文文案
