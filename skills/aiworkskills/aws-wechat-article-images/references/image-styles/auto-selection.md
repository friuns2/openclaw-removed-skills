# 自动推荐规则

当用户未指定风格时，根据文章内容自动推荐。

> 本文件主要用于**正文配图**的 Type 和 Style 自动推荐。封面风格的 fallback 见末尾「封面风格 fallback」节。

## 按文章调性（正文配图）

| 调性（config `tone`） | 推荐风格 |
|----------------------|---------|
| 正式、专业 | 扁平矢量、蓝图、商务 |
| 轻松、幽默 | 手绘、温暖、活泼 |
| 犀利 | 海报、极简 |

## 按文章内容信号

| 内容信号 | 推荐 Type | 推荐 Style |
|---------|-----------|-----------|
| 抽象概念、原理、隐喻 | concept | 扁平矢量、简约线条、手绘 |
| 教程、步骤、操作指南 | process | 扁平矢量、简约线条 |
| 知识框架、原理讲解、课程 | whiteboard | 手绘、简约线条 |
| 数据、统计、趋势、报告 | data-viz | 扁平矢量、商务 |
| 对比、评测、方案选择 | comparison | 扁平矢量、商务 |
| 架构、系统、模型、体系 | architecture | 蓝图、扁平矢量 |
| 知识全景、关系梳理、概览 | mindmap | 扁平矢量、简约线条 |
| 历程、里程碑、路线图 | timeline | 简约线条、商务 |
| Tips、要点列表、注意事项 | checklist | 扁平矢量、温暖 |
| 金句、核心观点、名言 | quote-card | 温暖、海报、极简 |
| 使用场景、产品展示、代入感 | scene | 温暖、水彩、手绘 |
| 故事、情感、散文、视觉呼吸 | atmosphere | 温暖、水彩、海报 |

## 按 config `article_category`（正文配图）

| 账号类型 | 默认风格基调 |
|---------|------------|
| 科技 | 蓝图 / 扁平矢量 |
| 职场 | 商务 / 简约线条 |
| 情感 | 温暖 / 水彩 |
| 教育 | 扁平矢量 / 手绘 |
| 品牌号 | 按品牌调性 |
| 个人IP | 温暖 / 手绘 |

## 配图密度

按 config `image_density`：

| 配置值 | 规则 |
|--------|------|
| 每节一图 | 每个 H2 小标题后一张 |
| 少图 | 仅封面 + 1-2 张关键配图 |
| 多图 | 每 2-3 段一张 |

writing 阶段已按此密度插入标记，images 阶段按标记执行即可。

## 封面风格 fallback

当 **`article.yaml` > `custom_cover_image_style` > `default_cover_image_style`** 合并后**无有效单选预设**（空列表或未收窄为单元素）时，Agent 从可用封面预设（扫描内置 + **`.aws-article/presets/cover-styles/`**）中按以下规则推荐：

| 调性 / 类型 | 推荐封面预设 |
|------------|------------|
| 科技、专业 | 科技风、简约 |
| 轻松、幽默 | 插画、漫画 |
| 犀利、观点 | 大字报 |
| 正式、商业 | 简约、品牌模板 |
| 通用 / 无明确信号 | 简约 |

注：推荐的预设名须在可用列表中存在。若推荐的预设不存在，从已有预设中择一最接近的。
