# 简约

## 描述

干净利落的极简设计，大面积留白，单一焦点元素，低饱和度配色。适合传递专业、可信赖的视觉印象。

## 适用场景

科技、商业、知识分享、专业分析类文章。适合正式或专业调性的账号。

## Prompt 要点

- minimal design, clean composition, ample whitespace, single focal point
- muted or monochromatic color palette, subtle gradients allowed
- geometric or abstract shapes preferred over literal illustrations
- no clutter, no decorative elements, breathing space around subject
- 画面内文字必须为中文，在 prompt 中直接写出要显示的中文文案
