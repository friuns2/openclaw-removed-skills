# 📊 零代码BI分析器 (data-insight-generator)

> 业务人员友好型数据分析引擎 | 自动清洗 → 指标推导 → 图表推荐 → 洞察输出

## 🎯 功能简介
无需编写 SQL 或掌握 BI 工具，只需粘贴数据片段并指定业务场景，AI 将自动完成数据质量诊断、核心指标计算、可视化方案匹配与可执行建议输出。适用于运营复盘、销售追踪、库存预警、用户增长分析等轻量场景。

## 🚀 快速开始
### 1. 准备数据
- 导出 CSV/JSON 数据前 10-50 行（含表头）
- 敏感字段（如手机号/身份证/真实姓名）请替换为脱敏占位符

### 2. 配置调用
```json
{
  "skill": "data_insight_generator",
  "inputs": {
    "data_content": "<表头+示例数据行>",
    "business_context": "销售",
    "analysis_goal": "评估各渠道投放ROI并优化预算分配"
  }
}
