# Augent

The audio & video layer for agents. 22 local MCP tools. No cloud, no API keys.

## Getting Started

1. Install augent via the ClawHub install button or `uv tool install augent`
2. FFmpeg is required: `brew install ffmpeg` (macOS) or `apt install ffmpeg` (Linux)

## Tools

| Tool | Description |
|------|-------------|
| `download_audio` | Download audio from video URLs (1000+ sites) |
| `transcribe_audio` | Transcribe with timestamps in 99 languages |
| `search_audio` | Keyword search with timestamps and context |
| `deep_search` | Semantic search by meaning |
| `search_memory` | Search across all stored transcriptions |
| `take_notes` | Download, transcribe, and save formatted notes |
| `clip_export` | Export a video clip for a time range |
| `chapters` | Auto-detect topic chapters |
| `search_proximity` | Find where two keywords appear near each other |
| `identify_speakers` | Speaker diarization |
| `separate_audio` | Isolate vocals from music and noise |
| `batch_search` | Search multiple files in parallel |
| `text_to_speech` | Text to speech with 54 local voices |
| `highlights` | Export best moments or topic-matched clips |
| `visual` | Extract frames from video at key moments |
| `tag` | Organize transcriptions with topic tags |
| `rebuild_graph` | Build Obsidian knowledge graph |
| `spaces` | Download or live-record X/Twitter Spaces |
| `list_files` | List media files in a directory |
| `list_memories` | Browse stored transcriptions |
| `memory_stats` | View memory statistics |
| `clear_memory` | Clear stored transcriptions |
