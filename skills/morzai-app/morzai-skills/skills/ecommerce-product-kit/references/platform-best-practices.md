# Platform Best Practices (Summary)

This file provides high-level guidance only.

## Core Principles
- Keep product readability first.
- Match image style to platform context.
- Prioritize compliance for marketplace hero images.

## Platform Notes
- Amazon: hero image should remain compliance-first and distraction-free.
- Social channels: prioritize authentic context and storytelling.
- DTC storefronts: balance brand feel with conversion clarity.

## Execution Notes
- Use a consistent ratio and lighting language across a set.
- If platform is unclear, default to conservative ecommerce-safe styling.
