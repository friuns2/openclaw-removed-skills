# Output Spec (Summary)

This file defines output boundaries at a high level.

## Output Types
- Hero image: first-click clarity and primary product focus.
- Detail image: materials, structure, and benefit explanation.
- Lifestyle image: real-world context and desirability.
- Decision image: information that reduces purchase hesitation.
- Marketing poster: campaign communication asset.
- Try-on / apparel visual: fit and on-body presentation.
- Listing set: full P1-P7 sequence.

## Selection Rule
- Single image request: choose one output type by goal.
- Full bundle request: use listing set workflow.
