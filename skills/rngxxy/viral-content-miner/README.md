# viral-content-miner

热点分析与爆款挖掘工具。用于快速定位全网热门话题、拆解爆款规律、输出可执行的创作方向，适合内容创作者、运营、品牌和策略团队。

## 这个 Skill 能帮你做什么

- 全网热点扫描：聚合多平台近期高热话题与趋势
- 指定话题深挖：分析某个事件/赛道/品牌的热度与爆款内容
- 爆款规律提取：总结高频关键词、情绪词、内容类型分布
- 平台差异对比：识别不同平台的内容偏好和爆款门槛
- 创作角度推荐：输出空白角度、差异化角度、人群细分角度

## 适用场景

- 选题策划：不知道最近写什么、拍什么
- 热点追踪：需要持续观察某个话题热度变化
- 竞品分析：拆解品牌/事件为什么火
- 方向判断：评估某赛道是否还有内容机会

## 触发示例（可直接复制）

- 全网最新的话题有哪些？
- 最近有什么热点可以做内容？
- 帮我分析一下 `XX` 话题热度
- `XX` 最近有哪些爆款内容？
- 给我 `XX` 方向的选题建议
- `XX` 在抖音和小红书哪个更容易起量？

## 覆盖平台

优先覆盖：抖音、小红书、公众号、快手、B 站  
补充覆盖：微博、视频号、知乎  
趋势验证：新闻门户、搜索趋势渠道

## 输出内容（交付物）

运行后通常会提供以下模块：

1. 关键词云报告（核心词、关联词、高频词）
2. 爆款作品库（标题、摘要、平台、时效、互动数据、来源链接）
3. 平台热度分析（平台对比与平台适配建议）
4. 创作角度建议（按潜力评级排序）
5. 爆款规律总结（标题结构、情绪词、内容公式）

## 工作方式（简版流程）

输入话题 -> 关键词提取与扩展 -> 多平台搜索 -> 数据验证与清洗 -> 聚合分析 -> 创作建议输出

## 数据与质量标准

- 默认优先近 30 天内的热点内容
- 强调时效性验证（优先 24h / 3d / 7d）
- 要求尽量提供可验证来源链接
- 对待验证数据会明确标注，避免误导

## 使用建议

- 输入越具体，结果越可执行（例如：`AI 智能体出海营销` > `AI`）
- 可加目标平台和目标人群（例如：`小红书 + 新手宝妈`）
- 可直接要求输出格式（例如：`只给我 TOP10 选题 + 3 条标题模板`）

## 注意事项

- 搜索结果是时点快照，热度会随时间变化
- 部分平台互动数据可能无法完全精确获取
- 当实时搜索不可用时，会给出降级说明与替代建议

## 示例输入

### 示例 1：全网热点
`帮我看下全网最近最火的 5 个话题，并给每个话题 2 个创作角度。`

### 示例 2：指定话题
`分析“户外露营”近 30 天在抖音、小红书、公众号的爆款内容，并给出差异化选题。`

### 示例 3：内容定位
`我想做职场成长方向，帮我判断现在还有哪些蓝海切入点。`
