# 内容类型模板参考

写文章前先判断选题适合哪种内容类型，同一选题用不同类型写出来效果完全不同。**每次只选一种类型为主线，不要混用**。

---

## 类型一：🎬 故事叙事型
**适合**：有事件、有人物、有转折的选题
**特点**：第一人称或第三人称叙事，有代入感，像看短篇小说

**结构模板**：
```html
<h2>🌸 一、[悬念开场]</h2>
<p>🌼 [场景描写，第1段必须抛出悬念]</p>
<h2>🍜 二、[情节发展]</h2>
<p>🌼 [叙事推进 + 冲突/对比]</p>
<h2>🔥 三、[高潮/反转]</h2>
<p>🌼 [全文情绪最高点]</p>
<h2>📍 四、[实用信息]</h2>
<p>🌼 [地点、交通、人均]</p>
```

**开头写法**：
```
"我第三次路过那家店的时候，终于忍不住推门进去了。
老板看了我一眼说：'又来了？这次终于肯点招牌菜了？'"
```

---

## 类型二：📋 清单攻略型
**适合**：目的地合集、美食榜单、必吃必游清单
**特点**：信息密度高，收藏价值强

**结构模板**：
```html
<h2>🌸 一、[总领全文 + 亮点预告]</h2>
<p>🌼 [开头给出核心价值点]</p>
<h2>🍜 二、[清单主体]</h2>
<p>🌼 [序号+地名：[上榜理由]]</p>
<p>🥟 · 地址：XXX · 人均：XX元 · 必吃/必玩：XXX</p>
```

**开头写法**：
```
"刷了三天小红书、问了五个本地朋友、实地踩了七条街，
终于整理出这份『XX目的地不完全避坑指南』。"
```

---

## 类型三：⚡ 对比揭秘型
**适合**：网红店打假、热门vs小众、种草vs拔草
**特点**：制造认知冲突，引发评论区辩论

**结构模板**：
```html
<h2>🌸 一、[抛出普遍认知]</h2>
<p>🌼 "大家都在说……"</p>
<h2>🍜 二、[揭秘真实情况]</h2>
<p>🌼 事实是…… / 但本地人却说……</p>
<h2>🔥 三、[给出参考答案]</h2>
<p>🌼 我的建议是……</p>
```

---

## 类型四：🌿 沉浸体验型
**适合**：目的地探访、美景美食体验、情绪型内容
**特点**：画面感极强，文字即画面

**开头写法**：
```
"下午三点的阳光从老槐树的缝隙里漏下来，
落在青石板路上，碎成一地金币。
远处传来豆腐坊的吆喝声，混着锅里油滋滋的响。
这一刻，我突然明白什么叫'值得专门来一趟'。"
```

---

## 类型五：💬 人物访谈型
**适合**：店主故事、本地人生活、专家推荐
**特点**：有声音、有态度，读者像在听真人说话

**对话写法**：
```html
<p>老板叫老陈，在这开店三十年了。我问他为啥不多开几家分店，
他说：<q>"开那么多干嘛？顾不过来，品质就散了。"</q></p>
```

---

## 类型六：📊 数据盘点型
**适合**：年度总结、热门榜单、趋势分析
**特点**：用数字说话，有理有据

**开头写法**：
```
"我统计了平台上XX篇打卡笔记、问了XX个本地人、实地去了XX个地方，
得出了一个结论：……"
```
