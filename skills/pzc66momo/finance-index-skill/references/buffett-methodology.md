# 沃伦·巴菲特 (Warren Buffett) 投资方法论与决策框架蒸馏

> 本文件由 女娲 (Nuwa Skill) 执行多源信息调研与思维框架提炼生成。
> 重点聚焦：巴菲特的决策框架（HOW he decides）与判断标准（His Filters）。

## 1. 核心心智模型 (Mental Models)

### 模型1：能力圈 (Circle of Competence)
- **一句话描述**：不要试图跨越10英尺的栏杆，去寻找1英尺的、你可以轻松跨越的栏杆。
- **判断标准**：你是否能用简单的语言向外行解释这家公司是如何赚钱的？你是否能大致看清它10年后的样子？
- **应用方式**：面对热门科技股、复杂的金融衍生品、或看不懂的重组并购时，直接归入“太难”的废纸篓。
- **局限性**：可能错过巨大的时代技术红利（如早期的亚马逊、谷歌），但能保住本金。

### 模型2：经济护城河 (Economic Moat)
- **一句话描述**：一家真正伟大的公司必须有一道坚固的“护城河”来保护它的高资本回报率。
- **判断标准**：
  - 是否拥有定价权（提高价格而不会流失客户）？
  - 护城河来源：无形资产（可口可乐、苹果）、转换成本（银行、SaaS）、网络效应（运通、Visa）、成本优势（Geico、BNSF）。
- **应用方式**：在筛选长期持有标的时，优先看公司的ROE（投入资本回报率）是否能长期维持在15%以上且不需要大量增量资本。

### 模型3：所有者收益与自由现金流 (Owner Earnings)
- **一句话描述**：一家公司的价值等于其存续期间所能产生的自由现金流的折现。
- **判断标准**：净利润 + 折旧/摊销 - 维持当前竞争地位所需的资本支出。
- **应用方式**：不看华尔街炒作的EBITDA，只看企业真正能装进口袋、能用于分红或回购的真金白银。

### 模型4：市场先生 (Mr. Market)
- **一句话描述**：市场是来为你服务的，而不是来指导你的。
- **判断标准**：市场的报价是情绪驱动的（时而极度抑郁，时而极度狂热）。
- **应用方式**：利用市场的极端情绪。在别人贪婪时恐惧，在别人恐惧时贪婪。当优质资产因为宏观恐慌被抛售时，重仓买入。

### 模型5：机构盲从症 (Institutional Imperative)
- **一句话描述**：大部分CEO会像旅鼠一样盲目模仿同行的行为，无论多愚蠢。
- **判断标准**：管理层是否热衷于昂贵且无协同效应的并购？是否通过做账粉饰业绩？
- **应用方式**：只投资那些诚实、理性配置资本、把股东当合伙人的管理层。

## 2. 决策启发式 (Decision Heuristics / 快速过滤规则)

1. **20个打孔位法则**：如果你一生只有20次投资机会，你会对每一次出手极其谨慎。如果一个标的你不想持有10年，那就连10分钟都不要持有。
2. **通胀期生存准则**：在通胀时期，最好的企业是那些只需要很少增量资本就能增加营业额的企业（如品牌消费品），最差的是需要大量资本投入才能维持运营的重资产企业（如航空、公用事业）。
3. **“太难”过滤器**：巴菲特桌上有个“太难”（Too Hard）的托盘。遇到看不懂商业模式、或者财务报表故意搞得很复杂的公司，直接扔进去，绝不浪费时间。
4. **一票否决制**：只要管理层出现过一次对股东不诚信的行为，无论公司再便宜，直接一票否决。

## 3. 表达DNA与风格特征

- **修辞偏好**：极度偏爱使用**生活化的隐喻**。把复杂的金融概念转化为日常事物（如“厨房里的蟑螂不止一只”、“退潮后才知道谁在裸泳”、“找好老婆比找好股票重要”）。
- **语气感**：从容、幽默、自嘲。常常拿自己过去的失败（如收购伯克希尔哈撒韦纺织厂、早年买入全美航空）开玩笑。
- **极简主义**：不使用华尔街黑话（EBITDA、Beta、Alpha、期权定价），坚持用四年级学生能听懂的语言写致股东信。

## 4. 反模式 (绝不会做的事)

- **绝不预测宏观**：从不基于对利率、宏观经济、地缘政治的预测来做投资决策。
- **绝不使用杠杆**：绝不借钱炒股，为了避免在极端市场行情下被强制平仓（哪怕概率只有1%）。
- **绝不跟风**：不参与自己看不懂的IPO或热门概念炒作（如早期的互联网泡沫、加密货币）。
- **绝不频繁交易**：查理和他的主要利润来源于“等待”，而不是买卖。

## 5. 诚实边界 (Skill的局限性)

- **盲区**：巴菲特的框架天然排斥那些需要烧钱扩张且初期不盈利的高增长科技公司。用这套框架，你必然会错过下一个亚马逊或特斯拉。
- **资金体量差异**：目前的巴菲特因为资金体量太大，只能买“大象”，其近年来的决策框架已经异化，普通散户生搬硬套其近期的重仓股（如西方石油、某些日本商社）可能并不具备同样的资金成本优势和安全垫。
