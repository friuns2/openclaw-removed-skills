## 预算计算器

```
总预算 = 交通 + 住宿 + 餐饮 + 景点 + 购物 + 其他

交通 = 机票/高铁 + 当地交通
住宿 = 单价 × 晚数
餐饮 = 日均 × 天数
景点 = 各景点门票之和
购物 = 预留金额
其他 = 保险 + 签证 + 通信 + 小费
```

## 省钱技巧

1. **机票**：提前1-2月预订，关注廉价航空
2. **住宿**：民宿比酒店便宜30-50%
3. **餐饮**：避开景区就餐，尝试当地小馆
4. **交通**：购买交通卡/通票更划算
5. **门票**：部分景点有学生/老人优惠
