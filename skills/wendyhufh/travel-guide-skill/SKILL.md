# Travel Guide Skill - 旅行攻略生成器

## 快速开始

```
/travel-guide-skill
/travel-guide-skill 去日本京都旅游，5天4晚，2个年轻人，预算1.5万
/travel-guide-skill 目的地：三亚，时间：7月中旬，亲子游(2大1小)，预算8000
```

## 功能

1. **交互式信息收集** - 通过问答引导用户完善旅行需求
2. **智能攻略生成** - 基于需求生成多种可选方案
3. **实时信息查询** - 天气预报、机票酒店价格参考
4. **个性化推荐** - 根据出行人群（年轻人/亲子/老年团）定制内容

## ⚠️ 内容真实性要求

**关键原则：所有具体数据必须来自实时查询，禁止编造**

### 必须查询的数据
- ✈️ 机票价格、航班号、时刻
- 🏨 酒店价格、名称、位置
- 🎫 景点门票价格、开放时间
- 🌤️ 天气预报
- 🍜 推荐餐厅名称（需验证存在）

### 可以估算的数据（需明确标注"估算"）
- 📊 总预算范围（基于查询到的单价计算）
- ⏱️ 行程时间安排
- 🚶 游览时长

### 数据标注格式
```
机票价格：¥2,800往返（来源：携程，查询时间：2024-04-01）
景点门票：¥400（来源：清水寺官网）
酒店均价：¥600-800/晚（来源：Booking.com，查询时间：2024-04-01）
```

### 查询失败处理
如果无法查询到实时数据，必须：
1. 明确告知用户"以下价格为参考价，建议自行核实"
2. 提供查询渠道供用户自行确认
3. 不要编造具体的航班号、餐厅名等

### 来源可信度优先级
1. 官方网站（景区官网、航司官网）
2. 主流平台（携程、飞猪、Booking、大众点评）
3. 攻略平台（马蜂窝、小红书、TripAdvisor）
4. 通用搜索结果（需交叉验证）

## 攻略内容

每个攻略方案包含：
- **机票/酒店** - 推荐航班、住宿选择及预估价格
- **游玩计划** - 每日行程安排、景点推荐
- **交通方式** - 当地交通建议、机场到市区方案
- **特色美食** - 必吃美食、推荐餐厅
- **天气预报** - 目的地温度、降雨、穿衣建议
- **注意事项** - 签证、小费、习俗等提醒

## 流程

### 步骤 1: 信息收集

**必填信息**（通过问答引导收集）：

| 信息项 | 示例 | 问题模板 |
|--------|------|----------|
| 目的地 | 日本京都 | "请问目的地是哪里？" |
| 出发地 | 北京 | "请问从哪里出发？" |
| 旅行日期 | 2024.04.15-04.20 | "请问计划什么时候出行？" |
| 天数 | 5天4晚 | "计划玩几天？" |
| 人数 | 2人 | "一共几个人出行？" |
| 人群类型 | 年轻人/亲子/老年团 | "出行人员情况是？（年轻人/亲子/老年团）" |
| 预算 | 1.5万人民币 | "预算大概是多少？" |

**可选信息**：
- 偏好风格（休闲/特种兵/文化体验/自然风光）
- 住宿偏好（民宿/酒店/青旅）
- 美食偏好（当地特色/中餐/不忌口）
- 特殊需求（无障碍/素食/宠物同行）

**等待用户确认后**继续下一步。

---

### 步骤 3: 信息查询

**⚠️ 重要：必须使用 WebSearch 和 WebFetch 工具查询实时数据，禁止编造**

**查询内容**：
1. **天气预报** - 目的地旅行日期的天气
2. **机票价格** - 出发地到目的地的航班参考
3. **酒店推荐** - 目的地住宿选择
4. **景点信息** - 热门景点、开放时间、门票
5. **美食推荐** - 特色美食、推荐餐厅
6. **交通方式** - 当地交通攻略

**查询示例**：
```
WebSearch: "京都 4月 天气 穿衣建议"
WebSearch: "北京到京都 机票 价格 2024年4月"
WebSearch: "京都酒店推荐 清淡型"
WebSearch: "京都必去景点 推荐路线"
WebSearch: "京都特色美食 必吃"
```

**查询结果处理**：
- ✅ 优先使用官方来源（景区官网、航司官网）
- ✅ 记录查询时间和来源
- ✅ 交叉验证关键信息（价格、开放时间）
- ⚠️ 若无法查询到实时数据，明确告知用户并建议自行核实
- ❌ 禁止编造航班号、餐厅名、价格等具体信息

---

### 步骤 4: 生成攻略方案

根据人群类型，提供 2-3 个不同风格的方案：

**方案类型**：

| 人群类型 | 推荐方案风格 |
|----------|--------------|
| 年轻人 | 特种兵打卡、文艺慢游、深度体验 |
| 亲子游 | 乐园优先、轻松休闲、教育体验 |
| 老年团 | 舒适养生、经典景点、节奏舒缓 |
| 情侣 | 浪漫氛围、美食之旅、网红打卡 |
| 独行 | 社交青旅、深度探索、背包客风格 |
