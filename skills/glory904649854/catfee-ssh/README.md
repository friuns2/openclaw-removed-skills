# Catfee SSH

SSH 远程服务器密码连接技能，让 AI Agent 通过密码认证连接远程服务器执行运维操作。

## 功能特点

- 🔐 **密码认证连接** - 无需配置 SSH 密钥，直接使用用户名密码连接
- 🖥️ **远程命令执行** - 支持执行任意 Shell 命令
- 📊 **服务诊断** - Nginx、Docker、系统服务等状态检查
- 📁 **文件操作** - 查看文件内容、目录结构
- 📝 **日志查看** - 实时查看服务日志

## 安装

```bash
clawhub install catfee-ssh
```

## 使用方法

当需要 SSH 连接服务器时，提供以下信息：

- **IP 地址**: 服务器 IP
- **用户名**: SSH 用户名（如 root）
- **密码**: SSH 密码
- **端口**: SSH 端口（默认 22）

### 示例对话

```
用户: 帮我连接这个服务器看看 nginx 状态
      IP: 192.168.1.100
      用户: root
      密码: yourpassword

Agent: [使用 catfee-ssh skill 连接服务器并执行 nginx -t 等诊断命令]
```

## 支持的操作

### 系统信息
- 查看主机名、系统版本
- 查看资源使用情况（CPU、内存、磁盘）

### 服务管理
- Nginx 配置检查、状态查看
- Docker 容器状态、日志查看
- Systemd 服务状态

### 文件操作
- 查看文件内容
- 列出目录结构
- 查看日志文件

### 网络诊断
- 查看端口监听状态
- 查看网络连接

## 技术实现

基于 PowerShell 的 Posh-SSH 模块实现，支持 Windows 环境。

### 依赖

- PowerShell 5.1+
- Posh-SSH 模块（自动安装）

## 安全说明

- ⚠️ 凭证不存储，每次使用时由用户提供
- ⚠️ 操作完成后自动关闭 SSH 会话
- ⚠️ 不建议在生产环境长期使用密码认证，推荐 SSH 密钥
