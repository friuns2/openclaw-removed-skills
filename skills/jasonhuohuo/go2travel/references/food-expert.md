# 美食专家 — 搜索参考

## 核心命令

```bash
# 搜索目的地美食
flyai keyword-search --query "{城市} 美食推荐 必吃"
flyai keyword-search --query "{城市} 特色小吃 美食街"
flyai keyword-search --query "{城市} 餐厅推荐 {菜系}"

# 搜索美食套餐
flyai keyword-search --query "{城市} 美食套餐 餐厅"
```

## 热门城市美食速查

| 城市 | 必吃美食 | 美食街区 |
|------|---------|---------|
| 成都 | 火锅、串串、担担面 | 春熙路、宽窄巷子、锦里 |
| 重庆 | 火锅、小面、酸辣粉 | 解放碑、洪崖洞、磁器口 |
| 广州 | 早茶、烧鹅、肠粉 | 上下九、北京路、西关 |
| 西安 | 肉夹馍、凉皮、羊肉泡馍 | 回民街、永兴坊 |
| 长沙 | 臭豆腐、糖油粑粑 | 坡子街、太平街 |
| 杭州 | 西湖醋鱼、东坡肉 | 河坊街、南宋御街 |
| 北京 | 烤鸭、炸酱面 | 王府井、牛街、簋街 |

## 输出要求

- 美食推荐必须包含人均价格
- 有图片必须展示 `![]({picUrl})`
- 标注口味特点（辣/清淡/甜）
- 注明是否需要排队/预订
