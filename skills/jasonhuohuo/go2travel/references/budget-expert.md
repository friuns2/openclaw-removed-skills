# 省钱专家 — 搜索参考

## 核心命令

```bash
# 搜索优惠套餐
flyai keyword-search --query "{目的地} 优惠 套餐 折扣"
flyai keyword-search --query "{目的地} 酒店+机票 套餐"
flyai keyword-search --query "{目的地} 特价 机票"

# 酒店低价搜索
flyai search-hotel --dest-name "{城市}" --sort price_asc --max-price 500

# 航班低价搜索
flyai search-flight --origin "{出发地}" --destination "{目的地}" \
  --dep-date "{日期}" --sort-type 3 --max-price 2000
```

## 省钱策略速查

### 机票
- **提前预订**：国内 2-4 周，国际 1-3 月 → 省 20-40%
- **错峰出行**：避开周五/周日，选周二/周三 → 省 15-30%
- **中转航班**：选择中转而非直飞 → 省 20-50%
- **红眼航班**：早班/晚班 → 省 10-25%

### 酒店
- **套餐购买**：酒店+门票套餐 → 省 15-35%
- **连住优惠**：连住 2 晚以上 → 省 10-20%
- **非核心区**：地铁沿线而非景区旁 → 省 20-40%

### 景点
- **联票/套票**：多景点联票 → 省 15-30%
- **提前网购**：线上购票比现场 → 省 5-15%

## 输出要求

- 价格对比必须用表格
- 标注"基于飞猪实时数据"
- 特价票注明退改限制
