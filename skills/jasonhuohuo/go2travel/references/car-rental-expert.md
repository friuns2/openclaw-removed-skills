# 租车专家 — 搜索参考

## 核心命令

```bash
# 综合搜索租车
flyai keyword-search --query "{城市} 租车自驾 {天数}天"

# 按车型搜索
flyai keyword-search --query "{城市} 租车 SUV 经济型"
flyai keyword-search --query "{城市} 租车 商务车 7座"
flyai keyword-search --query "{城市} 租车 新能源"
flyai keyword-search --query "{城市} 房车租赁"
```

## 车型推荐速查

| 场景 | 推荐车型 | 参考价格/天 |
|------|---------|------------|
| 城市代步（1-2人） | 经济型（朗逸/卡罗拉） | ¥100-200 |
| 周边自驾（2-4人） | 紧凑型SUV（逍客/哈弗H6） | ¥200-350 |
| 长途自驾（4-6人） | 中型SUV（汉兰达/途观L） | ¥350-500 |
| 商务出行 | 商务车（GL8/奔驰V级） | ¥500-1000 |
| 豪华体验 | 豪华车（BBA/保时捷） | ¥800-2000 |
| 房车旅行 | 房车 | ¥300-800 |

## 租车注意事项

- 异地还车通常有额外费用（¥200-500）
- 芝麻信用分 ≥ 650 可免押金
- 取车时拍照记录车身划痕
- 满油取还，否则收高额油费
- 山区/雪地建议选 SUV + 全险

## 输出要求

- 必须展示车型图片
- 标注取车点位置和营业时间
- 保险建议用表格对比
- 注明"基于飞猪实时数据"
