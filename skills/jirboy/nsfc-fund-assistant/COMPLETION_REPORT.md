## 🔧 统一入口核心特性

### 1. 全流程覆盖（6 阶段）
```
立项依据 → 研究内容 → 研究方案 → 特色创新 → 研究基础 → 形式审查
  ⭐⭐⭐        ⭐⭐⭐        ⭐⭐        ⭐⭐⭐        ⭐⭐        ⭐
```

### 2. 写作模板库
- **摘要** - 400 字以内，6 要素结构
- **立项依据** - 倒金字塔结构，6 段式
- **研究内容** - 3-4 条，逻辑递进
- **关键科学问题** - 2-3 个，理论 + 方法
- **创新之处** - 2-3 点，理论/方法/应用

### 3. 三级审查体系
- **P0** - 必须修改（影响提交）6 项
- **P1** - 重要修改（影响评审）6 项
- **P2** - 优化建议（提升质量）5 项

### 4. 文献检索策略
- 文献零编造原则
- 近 5 年≥30%
- 包含权威学者工作
- 包含最新研究进展

### 5. 时间节点管理
```
T-60 天 → 确定选题，开始文献调研
T-45 天 → 完成立项依据初稿
T-30 天 → 完成研究内容/方案
T-20 天 → 完成创新点/研究基础
T-14 天 → 第一次形式审查（P0/P1）
T-7 天  → 修改完善，第二次审查
T-3 天  → 最终检查，准备提交
```

## ⚠️ 注意事项

### 基金申请特殊性
- **时间节点：** 每年 3 月提交，需提前 60 天准备
- **形式审查：** P0 项必须零问题，否则直接淘汰
- **文献真实：** 绝不允许编造文献，必须验证
- **限项规定：** 注意申请人和参与者的限项规则

### 协作原则
- **先理解后写作**：所有修改前必须充分理解用户意图
- **逐步推进**：将申请书分解为可管理的小任务
- **持续反馈**：每个阶段完成后向用户确认
- **版本控制**：重要修改前创建备份
- **进度可见**：使用任务跟踪工具

### 质量保证原则
- **文献零编造**：所有引用必须通过检索验证
- **方法有据**：所有算法、方法必须引用文献支持
- **术语准确**：使用领域标准术语和规范表达
- **数据真实**：所有实验数据必须真实可验证
