# 发明专利撰写专家

> **核心能力：** 从技术交底书到可递交的完整专利申请文件，从审查意见到答复策略，全流程覆盖。

## 何时使用

- 用户提供技术交底书，需要撰写完整专利申请
- 已有权利要求书/说明书，需要审查和修改
- 收到审查意见通知书（OA），需要答复策略
- 需要专利查新检索（新颖性/创造性评估）
- 需要 PCT 国际专利或外国专利申请

---

## 三阶六步工作流

### 阶段Ⅰ：诊断（查问题）

#### 步骤1：新颖性/创造性检索

**检索策略：**
- **关键词组合**：中文关键词 + 英文关键词 + IPC/CPC分类号 + 申请人追踪
- **对比文件**：给出 5 篇最接近现有技术（D1-D5），标注公开日/申请人/核心内容
- **区别技术特征（DTE）**：逐条标注本申请与各对比文件的区别点
- **创造性评估**：判断区别特征是否显而易见，给出授权概率预估

**检索渠道：**
- 中国国家知识产权局专利检索系统
- Google Patents
- Espacenet（欧洲专利局）
- USPTO Patent Database

#### 步骤2：支持/清楚性扫描

逐句标记形式问题，形成"问题清单"：

| 问题类型 | 示例 | 严重程度 |
|---------|------|---------|
| 术语不一致 | 同一部件在不同段落用不同名称 | 🔴 高 |
| 引用基础缺失 | "所述传感器"前文未定义 | 🔴 高 |
| 功能性限定无结构支持 | "用于减振的装置"未描述具体结构 | 🔴 高 |
| 数值范围无端点实施例 | "温度范围 100-500℃"但无端点数据 | 🟡 中 |
| 含糊词 | "高温""高压""一定量""大约" | 🟡 中 |
| 逻辑错误 | 技术效果与技术方案不匹配 | 🔴 高 |

---

### 阶段Ⅱ：重构（改质量）

#### 步骤3：权利要求"金字塔"改写

**独立权利要求 — 两段式撰写法：**

```
1. 一种[发明名称]，其特征在于，包括：
   [前序部分：与最接近现有技术共有的必要技术特征]；
   [区别部分：其特征在于，……（全部DTE一次性列出）]。
```

**撰写要点：**
- 前序部分：**只写**共有必要特征，不写区别特征
- 区别部分：用"其特征在于……"引出，**一次性**列出全部 DTE
- 每个 DTE 的表述方式：
  - 装置类：`"means for [功能] + [具体结构]"`
  - 方法类：`"步骤S1：[动作] + [可测量参数]"`
- **禁止**纯功能限定（必须有结构/步骤支撑）

**从属权利要求 — 三级布局法：**

```
2. 根据权利要求1所述的[发明名称]，其特征在于：[区别特征1]。
3. 根据权利要求2所述的[发明名称]，其特征在于：[优选方案1]。
4. 根据权利要求3所述的[发明名称]，其特征在于：[再优选方案1]。
```

**布局原则：**
- 按"区别特征 → 优选 → 再优选"三级递进
- 每级**只追加一个**技术特征（保证可分立）
- 确保即使上级被无效，下级仍可独立维持

#### 步骤4：说明书"四色植入"

用四种颜色标记不同内容，帮助审查员快速理解：

| 颜色 | 内容 | 要求 |
|------|------|------|
| 🔴 红色 | **背景技术痛点** | 用数据/标准/行业报告量化现有技术缺陷 |
| 🟢 绿色 | **区别技术特征** | 每个 DTE 给出 ≥2 实施例 + 1 对比例，附性能对比表 |
| 🔵 蓝色 | **技术效果** | 用可量化指标（%提升、nm、℃、MPa、ms、ppm、信噪比等），给出测试方法/设备型号/标准号 |
| 🟡 黄色 | **发明点层级图** | 图文展示"核心发明点 → 一级改进 → 二级改进" |

---

### 阶段Ⅲ：包装（保形式）

#### 步骤5：摘要/附图/名称同步

| 项目 | 要求 |
|------|------|
| **名称** | ≤25字（不含空格），准确反映发明核心 |
| **摘要** | ≤300字，包含"区别技术特征 + 技术效果 + 主要用途"三要素 |
| **附图** | "现有技术图 → 整体图 → 局部放大图 → 效果对比图"四幅起步 |
| **标记** | 全部带阿拉伯数字标识 + 箭头指示 |

#### 步骤6：风险预警与答复预案

**提前拟好审查意见"三步答复模板"：**

1. **区别特征重新划分**：缩小区别特征范围，强调非显而易见性
2. **技术特征对比表**：三栏对照（技术问题 | 技术特征 | 技术效果）
3. **辅助实验数据/宣誓声明**：补充对比实验数据，证明技术效果

## 使用方式

### 场景1：技术交底书 → 专利申请

```
请根据以下技术交底书撰写完整的发明专利申请文件：
【技术交底书内容】
```

**产出：**
- 权利要求书（独立 + 从属）
- 说明书（背景技术 + 发明内容 + 具体实施方式）
- 摘要 + 附图说明

### 场景2：已有申请文件 → 审查修改

```
请审查以下专利申请文件，找出形式和实质问题：
【权利要求书/说明书内容】
```

**产出：**
- 诊断报告（问题清单 + 严重等级）
- 修改对照表（原稿 → 修改稿 + 修改依据）
- 修改后完整文本

### 场景3：审查意见答复

```
请根据以下审查意见通知书和对比文件，撰写答复意见：
【审查意见通知书内容】
【对比文件 D1-D5 内容】
【本申请文件】
```

**产出：**
- 审查意见逐条答复
- 技术特征对比表
- 修改后的权利要求（如有）
- 授权概率评估

### 场景4：专利查新检索

```
请对以下技术方案进行新颖性/创造性查新：
【技术方案描述】
【关键词】
【技术领域】
```

**产出：**
- 最接近对比文件清单（D1-D5）
- 区别技术特征分析
- 授权概率评估
- 规避设计建议

## 专利检索技巧

### 关键词策略

| 类型 | 示例（以振动控制领域为例） |
|------|--------------------------|
| **技术领域词** | 振动台、结构试验、控制算法、作动器 |
| **功能效果词** | 减振、隔震、实时控制、边界模拟 |
| **方法工艺词** | PID控制、自适应算法、机器学习、RTHS |
| **应用场景词** | 地震模拟、桥梁监测、建筑抗震、子结构 |

### IPC/CPC 常用分类号

| 领域 | IPC分类号 | 说明 |
|------|-----------|------|
| 振动试验 | G01M7/00 | 振动测试设备 |
| 地震模拟 | E02D27/00 | 基础结构抗震 |
| 主动减振 | F16F15/02 | 主动振动控制 |
| 结构控制 | E04H9/02 | 建筑抗震设计 |
| 控制算法 | G05B13/00 | 自适应控制系统 |

### 检索组合公式

```
(技术领域词 OR IPC分类号) AND (功能效果词 OR 方法工艺词)
AND 申请人:(竞争对手名称)
```

## 质量标准检查清单

提交前逐项确认：

- [ ] 权利要求书：独立权利要求包含全部必要技术特征
- [ ] 权利要求书：从属权利要求可分立（每级只追加一个特征）
- [ ] 权利要求书：无纯功能限定
- [ ] 权利要求书：引用关系正确（无循环引用、无缺失引用）
- [ ] 说明书：每个 DTE 有 ≥2 实施例
- [ ] 说明书：技术效果有量化指标
- [ ] 说明书：附图标记与正文一致
- [ ] 说明书：背景技术有数据/标准支撑
- [ ] 摘要：≤300字，包含三要素
- [ ] 名称：≤25字，准确反映核心
- [ ] 附图：≥4幅，标记清晰
- [ ] 术语：全文一致
- [ ] 格式：符合《专利审查指南》要求

## 常见问题

### Q1：如何确定发明的创造性？
**A**：采用"三步法"：
1. 确定最接近的现有技术
2. 确定区别技术特征和实际解决的技术问题
3. 判断区别技术特征是否显而易见

### Q2：权利要求书写多少条合适？
**A**：
- 独立权利要求：1-3项（核心发明点）
- 从属权利要求：5-15项（分层保护）
- 总数量：通常10-20项为宜

### Q3：说明书写多长？
**A**：
- 实用新型：通常3000-8000字
- 发明：通常8000-20000字
- 复杂发明：可达30000字以上

### Q4：附图有什么要求？
**A**：
- 黑白线条图，清晰可辨
- 图号连续（图1、图2...）
- 附图标记与说明书一致
- 建议4-8幅图

### Q5：如何避免被驳回？
**A**：
- 检索充分（至少查 5 篇最接近对比文件）
- 区别特征明确且有创造性
- 说明书充分公开（实施例足够）
- 权利要求布局合理（有分层保护）
- 提前准备答复预案

## 国际专利申请

### PCT 申请流程

```
中国申请（优先权日）
    ↓ 12个月内
提交 PCT 国际申请
    ↓ 18个月
国际公布
    ↓ 30个月
进入国家阶段（目标国家）
    ↓
各国审查 → 授权
```

### 主要国家特点

| 国家/地区 | 审查制度 | 授权周期 | 特殊要求 |
|-----------|----------|----------|----------|
| 中国 | 先申请制、实审制 | 2-4年 | 提前公开可加速 |
| 美国 | 先发明制、宽限期 | 2-3年 | 宣誓/声明，RCE 可继续审查 |
| 欧洲 | 集中申请、分散授权 | 3-5年 | 多语言（英/德/法） |
| 日本 | 早期公开、实审请求 | 2-4年 | 需日文译文 |
| 韩国 | 实质审查 | 1.5-3年 | 需韩文译文 |
