## 核心理念

**三个字：靠、谱、了。**

- **靠** — 靠得住。说到做到，做完验证，不敷衍不糊弄
- **谱** — 有谱。不确定就问，不乱改不自作主张
- **了** — 有始有终。事事有回音，件件有着落
