## 步骤 4：数据核验（5 分钟）

### 4.1 实时行情核验
- [ ] 两次调用数据一致
- [ ] 涨跌幅 = (最新价 - 昨收)/昨收×100%（误差<0.1%）
- [ ] 市值 = 最新价×总股本（误差<1%）

### 4.2 表格结构核验
- [ ] 所有表格无错行错列
- [ ] 表头与数据列对齐
- [ ] 合计/总计项计算正确

### 4.3 财务数据核验
- [ ] 主营收入合计 = 各业务板块收入之和（误差<0.1%）
- [ ] 毛利率 = (营收 - 成本)/营收×100%（误差<0.1pct）
- [ ] 同比 = (本期 - 上年同期)/上年同期×100%（误差<0.1%）
- [ ] 占比之和 = 100%（容差<0.1%）
- [ ] 毛利率 ≥ 净利率

### 4.4 机构评级核验
- [ ] 机构数量合计 = 覆盖机构总数
- [ ] 各评级占比之和 = 100%

### 4.5 目标价核验
- [ ] 最高价 ≥ 平均价 ≥ 中位数 ≥ 最低价
- [ ] 上涨空间 = (平均目标价 - 当前股价)/当前股价×100%（误差<0.1%）

### 4.6 逻辑一致性核验
- [ ] ROE 杜邦拆解平衡（净利率×周转率×权益乘数≈ROE，误差<1%）
- [ ] 资金流向与股价走势无明显的背离
- [ ] 业绩快报与盈利预测数据一致
