# 分析框架详解

## 一、杜邦分析框架

杜邦分析法将 ROE 拆解为三个核心驱动因素：

```
ROE = 净利率 × 资产周转率 × 权益乘数
```

### 1. 净利率（Profit Margin）
- **公式**：净利润 / 营业收入
- **含义**：反映产品盈利能力和成本控制水平
- **分析要点**：
  - 与同行业对比
  - 历史趋势分析（最近 5 期）
  - 毛利率与净利率差值分析（费用控制）

### 2. 资产周转率（Asset Turnover）
- **公式**：营业收入 / 平均总资产
- **含义**：反映资产运营效率
- **分析要点**：
  - 应收账款周转率
  - 存货周转率
  - 与同行业对比

### 3. 权益乘数（Equity Multiplier）
- **公式**：平均总资产 / 平均股东权益
- **含义**：反映财务杠杆水平
- **分析要点**：
  - 资产负债率
  - 财务风险评估
  - 杠杆变动趋势

## 二、估值分析框架

### PE（市盈率）
- **适用行业**：消费、医药、TMT 等成长型行业
- **公式**：市值 / 净利润（TTM）
- **分析维度**：
  - 绝对值：与历史均值/中位数对比
  - 相对值：与同行业可比公司对比
  - 历史分位：当前 PE 在近 5 年的百分位

### PB（市净率）
- **适用行业**：银行、周期、重资产行业
- **公式**：市值 / 净资产（LF）
- **分析维度**：
  - 破净股（PB<1）分析
  - ROE 与 PB 的匹配度
  - 历史分位分析

### PEG（市盈率相对盈利增长比率）
- **公式**：PE / 净利润增长率
- **含义**：估值与增速的匹配度
- **判断标准**：
  - PEG < 1：估值可能低估
  - PEG = 1：估值合理
  - PEG > 1：估值可能高估

### PS（市销率）
- **适用场景**：尚未盈利的成长型企业
- **公式**：市值 / 营业收入（TTM）

## 三、产业链分析框架

### 产业链位置识别
| 位置 | 特征 | 典型行业 |
|------|------|----------|
| 上游 | 原材料供应、资源属性 | 矿产、石化、农业 |
| 中游 | 加工制造、组装 | 电子制造、机械加工 |
| 下游 | 品牌消费、渠道 | 食品饮料、零售 |

### 成本传导能力分析
- **强传导**：产品差异化高、议价能力强
- **中传导**：行业竞争激烈、部分传导
- **弱传导**：产品同质化、完全竞争市场

### 下游需求稳定性分析
- **高稳定性**：必需消费、基础设施
- **中稳定性**：可选消费、一般制造
- **低稳定性**：周期性强、依赖投资

## 四、资金流向分析框架

### 主力资金流向
- **当日净流入**：反映短期资金态度
- **近 5 日净流入**：反映中期趋势
- **近 20 日净流入**：反映长期趋势

### 北向资金分析
- **持仓变化**：外资持股增减
- **流入流出**：每日净买入/卖出
- **持仓占比**：外资持股比例

### 机构持仓分析
- **基金持仓**：公募/私募基金
- **社保持仓**：社保基金动向
- **险资持仓**：保险资金配置

### 融资融券分析
- **融资余额**：杠杆资金态度
- **融券余量**：做空情绪
- **两融占比**：融资融券占流通市值比例

## 五、技术分析框架

### 均线系统
- **多头排列**：MA5 > MA20 > MA60 > MA250（看涨）
- **空头排列**：MA5 < MA20 < MA60 < MA250（看跌）
- **金叉/死叉**：短期均线上穿/下穿长期均线

### MACD 指标
- **DIFF 与 DEA**：快线与慢线
- **金叉**：DIFF 上穿 DEA（看涨信号）
- **死叉**：DIFF 下穿 DEA（看跌信号）
- **红绿柱**：动能强弱

### RSI 指标
- **超买区**：RSI > 70（可能回调）
- **超卖区**：RSI < 30（可能反弹）
- **中性区**：30 ≤ RSI ≤ 70

### 布林线（Bollinger Bands）
- **上轨**：压力位
- **中轨**：20 日均线
- **下轨**：支撑位
- **收口/开口**：波动率变化

## 六、机构评级分析框架

### 评级类型定义
| 评级 | 含义 | 预期涨幅 |
|------|------|----------|
| 买入 | 强烈推荐 | >20% |
| 增持 | 推荐 | 10-20% |
| 维持 | 中性 | -10%~10% |
| 减持 | 谨慎 | -10%~-20% |
| 卖出 | 回避 | <-20% |

### 评级集中度分析
- **主流评级占比**：（买入 + 增持）/ 总数
- **评级分歧度**：不同评级间的分布离散程度
- **评级变化趋势**：近 3 月上调/下调数量

### 目标价分析
- **平均目标价**：算术平均值
- **中位数目标价**：避免极端值影响
- **上涨空间**：（平均目标价 - 当前价）/ 当前价
