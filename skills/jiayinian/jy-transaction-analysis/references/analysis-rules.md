# 分析规则详解

## 闭环交易识别

### 定义

闭环交易指在同一分析时段内，对同一证券代码同时存在买入和卖出记录的交易。

### 匹配规则

1. **证券代码匹配**：以 6 位证券代码为准进行匹配
2. **方向要求**：必须同时包含买入和卖出操作
3. **时间顺序**：买入日期必须早于卖出日期
4. **多次买卖**：若同一标的有多次买卖，按时间顺序配对（先进先出）

### 计算公式

```
持仓天数 = 卖出日期 - 买入日期（交易日）
实际盈亏金额 = 卖出成交金额 - 买入成交金额
实际盈亏比例 = (盈亏金额 / 买入成交金额) × 100%
```

### 持仓周期分类

| 类别 | 持仓天数 | 说明 |
|------|---------|------|
| 短线 | ≤ 5 个交易日 | 超短线操作 |
| 波段 | 6–20 个交易日 | 短线波段 |
| 中线 | > 20 个交易日 | 中线持有 |

## 关键交易筛选

### 筛选逻辑

从所有有效交易中筛选**不超过 10 笔**关键交易，按优先级顺序：

```
1. 闭环交易（全部纳入）
   ↓
   若数量 < 10，继续
   ↓
2. 极端行情日交易（全部纳入）
   ↓
   若数量 < 10，继续
   ↓
3. 按成交金额降序补足至 10 笔
```

### 极端行情日定义

沪深 300 指数单日涨跌幅达到以下标准的交易日：

| 级别 | 阈值 | 说明 |
|------|------|------|
| 一般极端 | ±5% 至 ±7% | 较为罕见的单日波动 |
| 严重极端 | ≥ ±7% | 重大事件驱动的大幅波动 |
| 历史罕见 | ≥ ±10% | 历史极少见的极端行情 |

> **注意**：本 Skill 将上述三个级别统称为"极端行情"，只要达到任一级别阈值即纳入关键交易。

### 示例

```
假设用户有 15 笔交易：
- 闭环交易：4 笔 → 全部纳入（累计 4 笔）
- 极端行情日交易：3 笔（其中 1 笔已在闭环中） → 纳入 2 笔新交易（累计 6 笔）
- 剩余非闭环交易：8 笔 → 按金额取前 4 笔（累计 10 笔，达到上限）
```

## 大盘背景分析

### 指数选择

默认使用 **沪深 300 指数**（代码：000300.SH）作为大盘基准。

### 时间窗口

对每笔关键交易，获取以下时间窗口的指数数据：

- **交易日前 5 个交易日**：判断买入时市场趋势
- **交易日后 5 个交易日**：判断卖出后市场走势
- **交易当日**：获取当日涨跌幅

### 市场阶段判断

| 条件 | 判断 |
|------|------|
| 区间涨幅 > 3% | 上行阶段 |
| 区间跌幅 > 3% | 下行阶段 |
| -3% ≤ 区间涨跌幅 ≤ 3% | 震荡阶段 |

## 个股行情分析

### 当日行情指标

| 指标 | 用途 |
|------|------|
| 开盘价、收盘价 | 判断成交价在当日价格区间的位置 |
| 当日涨跌幅 | 判断个股当日表现 |
| 成交量、换手率 | 判断市场活跃度 |

### 成交价位置判断

```
若 成交价 < (开盘价 + 收盘价) / 2 - 1% → 偏低
若 成交价 > (开盘价 + 收盘价) / 2 + 1% → 偏高
否则 → 居中
```

## 买入点分析

### 股价状态判断

获取交易日前 5 个交易日的价格走势：

| 条件 | 判断 |
|------|------|
| 收盘价持续上涨（5 日涨幅 > 3%） | 上涨通道（追涨） |
| 收盘价持续下跌（5 日跌幅 > 3%） | 下跌通道（抄底） |
| 涨跌幅在±3% 之间波动 | 横盘整理（盘整期介入） |

### 顺势/逆势判断

对比同期沪深 300 走势：

```
若 个股涨幅 > 指数涨幅 + 2% → 强势股，可能为顺势操作
若 个股跌幅 < 指数跌幅 - 2% → 弱势股，可能为逆势抄底
否则 → 跟随大盘
```

### 资金流向分析

获取买入日及前 3 个交易日的主力资金净流入：

| 条件 | 判断 |
|------|------|
| 连续 3 日主力净流入 | 主力建仓，聪明钱共振 |
| 连续 3 日主力净流出 | 主力出货，逆主力方向 |
| 净流入/流出交替 | 主力分歧 |

### 技术指标分析

| 指标 | 多头信号 | 空头信号 |
|------|---------|---------|
| MA5 vs MA20 | MA5 > MA20（金叉） | MA5 < MA20（死叉） |
| MACD | DIF > DEA（金叉） | DIF < DEA（死叉） |
| RSI | RSI < 30（超卖） | RSI > 70（超买） |

**左侧/右侧交易判断**：
- 左侧交易：股价下跌通道 + RSI 超卖 + 逆势买入
- 右侧交易：股价上涨通道 + MACD 金叉 + 顺势买入

## 卖出点分析

### 卖出后走势判断

获取交易日后 5 个交易日的价格走势：

| 条件 | 判断 |
|------|------|
| 卖出后涨幅 > 5% | 卖飞 |
| 卖出后跌幅 > 5% | 精准止盈 |
| 卖出后跌幅 > 3% 且为亏损卖出 | 精准止损 |
| 卖出后涨幅 > 3% 且为亏损卖出 | 过早止损 |
| 涨跌幅在±3% 之间 | 正常离场 |

### 操作质量评价

结合盈亏和卖出后走势：

| 盈亏 | 卖出后走势 | 评价 |
|------|-----------|------|
| 盈利 | 下跌 | 精准止盈 ✅ |
| 盈利 | 大涨 | 止盈偏早 ⚠️ |
| 亏损 | 下跌 | 止损有效 ✅ |
| 亏损 | 大涨 | 止损过早 ⚠️ |

## 行为模式归纳

### 交易集中度

```
若 前三大标的交易金额占比 > 70% → 集中型
若 前三大标的交易金额占比 < 40% → 分散型
否则 → 均衡型
```

### 板块偏好分类

| 板块 | 典型行业 |
|------|---------|
| 大盘蓝筹 | 银行、保险、券商、石油、电力 |
| 消费 | 食品饮料、白酒、家电、医药 |
| 科技成长 | 半导体、电子、计算机、通信 |
| 周期 | 钢铁、煤炭、有色、化工、建材 |
| 高端制造 | 新能源车、光伏、风电、机械 |

### 持仓风格判断

```
若 存在闭环交易：
  平均持仓天数 ≤ 5 → 短线风格
  平均持仓天数 6-20 → 波段风格
  平均持仓天数 > 20 → 中线风格
否则：
  标注"本片段内无完整买卖闭环，持仓风格暂无法判断"
```

### 资金共振特征

```
统计所有买入点的主力资金流向：
若 超过 70% 的买入点主力同步净流入 → 共振型
若 超过 70% 的买入点主力净流出 → 逆主力型
否则 → 混合型
```

## 风险点识别

### 常见风险点

| 风险类型 | 判断条件 | 建议 |
|---------|---------|------|
| 止损执行偏慢 | 平均亏损幅度 > 7% | 设置严格止损线 |
| 止盈过早 | 盈利交易平均收益率 < 3% | 适当放宽止盈 |
| 持仓过于集中 | 单一标的占比 > 40% | 分散投资 |
| 频繁交易 | 日均交易 > 3 笔 | 减少交易频率 |
| 追涨杀跌 | 买入点多在上涨通道高位 | 等待回调介入 |
| 逆主力操作 | 超过 70% 买入点主力净流出 | 关注资金流向 |

## 数据不足处理

### 降级策略

| 数据缺失类型 | 处理方式 |
|------------|---------|
| 大盘行情缺失 | 标注"⚠️ 大盘数据暂缺"，跳过市场阶段判断 |
| 个股行情缺失 | 标注"⚠️ 个股数据暂缺"，跳过成交价位置分析 |
| 资金流向缺失 | 标注"⚠️ 资金流向数据暂缺"，跳过资金共振分析 |
| 技术指标缺失 | 标注"⚠️ 技术指标数据暂缺"，跳过左侧/右侧判断 |
| 全部行情缺失 | 仅输出交易解析和闭环识别，其他章节标注数据缺失 |

### 原则

- **不编造数据**：所有分析必须基于真实获取的数据
- **透明标注**：明确告知用户哪些分析因数据缺失而跳过
- **保留可用部分**：即使部分数据缺失，仍输出可用的分析结果
