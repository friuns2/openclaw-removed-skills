# 顶级名医风格中医辨证Skill（最强版·全面升级）
## 核心功能（模拟顶级名医临证，全面升级）
1.  智能问诊：自动追问关键信息（汗、寒热、饮食、二便、舌象、脉象、月经（女性）、小儿发育（儿科）），补齐辨证依据，避免片面
2.  八纲辨证：精准判断阴阳、表里、寒热、虚实，明确病机核心，新增辨证逻辑校验，提升准确率
3.  体质匹配：结合九种体质，细化体质与证型的关联，调整方药适配个人体质，新增体质调理建议
4.  遣方用药：动态生成处方（药名+参考量级+炮制方法），遵循君臣佐使配伍原则，新增一病多方择优推荐
5.  典籍支撑：引用《黄帝内经》《伤寒论》《金匮要略》《温病条辨》等经典原文，附条文深度解读
6.  医案参考：引用历代名医（叶天士、张锡纯、张景岳、朱丹溪等）相关医案，每证型新增2-3则，辅助理解诊疗思路
7.  临证加减：根据兼症、体质差异、年龄（小儿/老人），给出灵活加减方案，真正因人而异、辨证施治
8.  医嘱禁忌：结合症状、体质、证型，细化饮食、作息、护理建议，新增用药禁忌（禁用/慎用）清单
9.  舌脉细化：新增舌象、脉象精准判断逻辑，结合中医舌脉理论，辅助提升辨证精准度
10. 多病症覆盖：新增内科、妇科、儿科常见证型，解决原有证型单一问题

## 调用方式
```bash
python3 scripts/tcm.py
```

## 输出规范（完全模拟名医临证口吻，全面升级）
1.  问诊记录（模拟名医问诊，整理关键症状，含女性月经、小儿发育等专项信息）
2.  八纲辨证结论（明确病机、病位、病性，新增辨证逻辑说明）
3.  体质判断（九种体质分型，说明体质对病情的影响及专属调理建议）
4.  治法（遵循“汗和下消吐清温补”八法，明确治疗核心，新增治法原理说明）
5.  处方（药名+参考量级+炮制方法，新增一病多方推荐及择优理由）
6.  君臣佐使解析（详细说明每味药的配伍意义，体现名医用药思路，新增配伍禁忌提示）
7.  典籍依据（经典原文+条文深度解读，说明处方来源及适配病机）
8.  历代名医医案参考（引用2-3则相关名医医案，佐证诊疗逻辑，新增医案与当前证型的关联分析）
9.  临证加减（分体质、分兼症、分年龄，给出具体加减方案，新增加减药物的用量建议）
10. 医嘱与禁忌（饮食、作息、护理建议，及用药禁忌清单，明确禁用/慎用药物）
11. 郑重安全提示（重复强调，规避合规风险，新增小儿、孕妇用药特殊提示）
