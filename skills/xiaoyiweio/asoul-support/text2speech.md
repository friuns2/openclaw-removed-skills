大家好呀！今天给大家介绍一款我做的 A-SOUL 全自动应援插件，叫 A-SOUL Support。

它可以帮我们自动检测成员开播、自动挂机涨亲密度、点亮粉丝牌、还有视频三连和动态点赞，基本上把日常应援的活儿都包了。

先带大家看一下这个项目的 GitHub 主页。

安装非常简单，如果你有 OpenClaw 的话，只需要复制这一句话发给它，就自动装好了。没有 OpenClaw 也没关系，后面会说怎么手动用。

功能这里列得很清楚，心跳挂机、粉丝牌点亮、投币、视频点赞、动态点赞，一共五个功能。需要开播的会自动检测，不需要开播的会定时执行。

安全方面大家放心，Cookie 是加密存储的，代码全部开源，不会乱投币不会关注陌生人，可以自己去看每一行代码。

如果你不用 OpenClaw，也可以 Fork 这个仓库，用 GitHub Actions 自动帮你视频点赞和动态点赞，设置方法在这里，跟着做就行。

好，接下来是重头戏，给大家看看实际效果！

第一个，直播挂机。

我们的插件每三十分钟会自动检测一次 A-SOUL 成员有没有在播。检测到开播之后，会先发十条弹幕帮你点亮粉丝牌，然后开始心跳挂机涨亲密度，一直挂到下播为止。

大家看，我打开我的 B站粉丝牌面板，贝拉的贝极星，今日亲密度三十，已经满了。这就是昨天自动挂机挂上来的，我自己完全没有打开过 B站。

第二个，自动通知。

我打开我的 Discord，大家看这里。插件检测到贝拉开播之后，会自动给我发一条消息，告诉我谁开播了、直播间叫什么。下播之后也会通知我，告诉我直播了多长时间。

然后每天早上还有一份挂机日报，把昨天谁开播了、挂了多久、亲密度涨了多少，全部汇总在一起，一目了然。

第三个，视频自动点赞。

这个跑在 GitHub Actions 上，每两天自动执行一次。我打开 GitHub 的 Actions 页面给大家看，最近几次执行全部是绿色的，都成功了。点进去可以看到详细的点赞记录，哪个成员的哪个视频，点赞成功还是之前已经赞过了，都写得很清楚。

第四个，也是我觉得最酷的，我可以直接在 Discord 发一条消息，让它帮我给乃琳最近七天的视频三连。

大家看，我发送命令，稍等一下，好，它回复我了，三连成功。我现在打开 B站验证一下，点进乃琳的主页，最近的视频，大家看，点赞、投币、收藏，全部到位了。

好啦，功能展示就到这里。

最后说两句心里话。做这个工具真的不是让大家躺平不看直播的。我自己是因为工作太忙，经常加班错过开播，看着粉丝牌一个个熄灭真的很心疼，才写了这个在忙的时候帮我守着。

如果你有时间的话，还是去直播间看直播吧，跟大家一起刷弹幕、跟嘉然贝拉她们互动，那种快乐是工具给不了的呀。

另外就算没有 OpenClaw，你也可以直接用 Python 跑脚本，一行命令就能开始挂机，具体用法在 GitHub 的 README 里都有。
