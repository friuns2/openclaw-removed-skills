# 租房避坑指南

## 一、租房前准备

### 预算规划
- **租金占比**：月租金不宜超过月收入的30%
- **隐性成本**：物业费、网费、水电燃气费、中介费、搬家费
- **押金准备**：通常为1-3个月租金

### 选址要点
- 通勤时间：建议单程不超过45分钟
- 交通便利：距离地铁站步行不超过10分钟为佳
- 生活配套：周边有超市、菜市场、医院、药店

## 二、看房检查清单

### 房屋本身
- [ ] 采光：白天看房，检查自然光线
- [ ] 通风：开窗测试空气流通
- [ ] 隔音：关窗听外界噪音，敲墙检查隔音
- [ ] 水电：测试所有插座、开关、水龙头
- [ ] 家具：检查家具是否完好，拍照记录瑕疵
- [ ] 电器：测试空调、热水器、冰箱等
- [ ] 网络：测试网速，确认网络费用
- [ ] 卫生间：检查水压、排水、是否有异味
- [ ] 门窗：检查是否能正常关闭上锁

### 小区环境
- [ ] 安全性：是否有门禁、保安、监控
- [ ] 噪音：是否靠近马路、工地、商铺
- [ ] 停车：如有车，确认停车位
- [ ] 快递：确认快递收发方式

## 三、合同签订要点

### 必须确认
- [ ] 房东身份：查看房产证和身份证，确认是本人
- [ ] 产权清晰：确认房屋无抵押、无纠纷
- [ ] 合同期限：明确起止日期
- [ ] 租金金额：大小写一致，明确支付方式
- [ ] 押金退还：明确退还条件和时间
- [ ] 费用承担：明确物业费、水电燃气费由谁承担

### 关键条款
- [ ] 维修责任：哪些维修由房东负责
- [ ] 转租规定：是否允许转租/合租
- [ ] 提前退租：违约金如何计算
- [ ] 续租条件：到期后如何续租、涨租规则
- [ ] 房屋交接：入住和退房的验收标准

### 签约注意事项
- 使用正规合同模板
- 所有口头承诺写入合同
- 保留合同原件
- 转账备注"房租"，保留凭证
- 入住前拍照/录像记录房屋现状

## 四、常见陷阱识别

### 虚假房源
- 价格明显低于市场价
- 图片过于精美，与描述不符
- 要求先交定金再看房
- 以各种理由推脱实地看房

### 黑中介套路
- 收取高额看房费
- 合同埋坑（高额违约金、不合理扣费）
- 虚假承诺（口头答应但不写入合同）
- 克扣押金（以各种理由不退押金）

### 二房东风险
- 未获原房东同意转租
- 卷款跑路
- 合同到期原房东收回房屋

### 费用陷阱
- 隐藏费用（卫生费、管理费、设施使用费）
- 水电费加价（高于正常价格）
- 退租时高额清洁费/维修费

## 五、入住后注意事项

- 及时更换门锁或密码
- 检查消防设施是否完好
- 了解邻居情况
- 加入小区业主群/租户群
- 保留房东联系方式和紧急联系方式
- 定期拍照记录房屋状况（每月一次）

## 六、维权途径

如遇纠纷：
1. 与房东协商
2. 向中介公司投诉
3. 拨打12345市民热线
4. 向住建委投诉
5. 报警或向法院起诉
