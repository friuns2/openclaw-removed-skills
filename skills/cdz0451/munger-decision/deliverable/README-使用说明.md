## 🎯 如何使用？

### 方式一：日常决策参考

**场景：** 面临重大决策时

**步骤：**
1. 打开 `models-full.json`
2. 浏览对应分类的模型
3. 问自己："这个模型能帮我看到什么盲点？"

**示例：**
> 投资决策前 → 查看"投资模型" + "心理学模型"
> 
> 问：我是否陷入了确认偏误？我是否在能力圈外操作？我有安全边际吗？

---

### 方式二：编程集成（开发者）

```javascript
const models = require('./models-full.json');

// 获取所有核心模型
const coreModels = models.filter(m => m.category === 'core');

// 搜索特定模型
const model = models.find(m => m.id === '07'); // 逆向思维
console.log(model.name); // "逆向思维"
console.log(model.desc); // "反过来想——通过思考如何失败来避免失败"
```

---

### 方式三：学习卡片（打印/导入 Anki）

每个模型包含：
- **名称**：如"逆向思维"
- **描述**：一句话解释
- **关键词**：便于搜索
- **分类**：所属学科领域

**建议：** 每天学习 1-2 个模型，40 天掌握全部 83 个

## 💡 使用建议

### ✅ 推荐做法
1. **每天 1 个模型** - 40 天掌握全部
2. **决策前查阅** - 问"哪个模型能帮到我？"
3. **写反思笔记** - 记录模型如何改变你的思考
4. **教给别人** - 费曼学习法，加深理解

### ❌ 避免做法
1. **一次性读完** - 记不住，用不上
2. **只看不练** - 模型需要实践内化
3. **生搬硬套** - 理解本质，灵活运用
