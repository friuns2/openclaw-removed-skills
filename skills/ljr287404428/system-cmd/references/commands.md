# 系统命令参考文档

## 支持的 Windows 系统命令

### 1. ipconfig
**命令**: `ipconfig /all`
**功能**: 显示所有网络适配器的完整 TCP/IP 配置
**输出包含**:
- 主机名
- 主 DNS 后缀
- 节点类型
- IP 路由是否启用
- WINS 代理是否启用
- 每个网络适配器的详细信息（DHCP、IP地址、子网掩码、默认网关、DNS服务器等）

**常用参数**:
- `/all`: 显示完整配置信息
- `/release`: 释放 DHCP 分配的 IP 地址
- `/renew`: 更新 DHCP 分配的 IP 地址
- `/flushdns`: 清除 DNS 解析器缓存

### 2. tasklist
**命令**: `tasklist`
**功能**: 显示当前在本地或远程计算机上运行的进程列表
**输出格式**:
```
映像名称                       PID 会话名              会话#       内存使用
========================= ======== ================ =========== ============
System Idle Process              0 Services                   0          8 K
System                           4 Services                   0      1,028 K
```

**常用参数**:
- `/S system`: 指定远程系统
- `/U username`: 执行命令的用户上下文
- `/P password`: 指定用户密码
- `/M module`: 列出使用给定 DLL 模块的进程
- `/V`: 显示详细任务信息

### 3. netstat
**命令**: `netstat -an`
**功能**: 显示协议统计信息和当前 TCP/IP 网络连接
**参数说明**:
- `-a`: 显示所有连接和侦听端口
- `-n`: 以数字形式显示地址和端口号
- `-b`: 显示创建每个连接或侦听端口所涉及的可执行文件
- `-o`: 显示拥有的与每个连接关联的进程 ID

**输出示例**:
```
活动连接

  协议  本地地址          外部地址        状态
  TCP    0.0.0.0:135           0.0.0.0:0              LISTENING
  TCP    0.0.0.0:445           0.0.0.0:0              LISTENING
  TCP    127.0.0.1:49667       127.0.0.1:49668        ESTABLISHED
```

### 4. systeminfo
**命令**: `systeminfo`
**功能**: 显示关于计算机及其操作系统的详细配置信息
**输出包含**:
- 主机名
- OS 名称
- OS 版本
- OS 制造商
- OS 配置
- OS 构建类型
- 注册的所有者
- 注册的组织
- 产品 ID
- 初始安装日期
- 系统启动时间
- 系统制造商
- 系统型号
- 系统类型
- 处理器
- BIOS 版本
- Windows 目录
- 系统目录
- 启动设备
- 系统区域设置
- 输入区域设置
- 时区
- 总物理内存
- 可用物理内存
- 虚拟内存: 最大值
- 虚拟内存: 可用
- 虚拟内存: 使用中
- 页面文件位置
- 域
- 登录服务器

## 命令执行注意事项

### 权限要求
1. **systeminfo**: 需要标准用户权限
2. **tasklist**: 需要标准用户权限，查看其他用户进程需要管理员权限
3. **netstat**: 需要标准用户权限
4. **ipconfig**: 需要标准用户权限，释放/更新 IP 需要管理员权限

### 执行时间
1. **systeminfo**: 执行时间较长（3-10秒）
2. **tasklist**: 执行速度快（1-3秒）
3. **netstat**: 执行速度快（1-3秒）
4. **ipconfig**: 执行速度快（1-3秒）

### 输出大小限制
1. **tasklist**: 输出可能较大（50-200行）
2. **netstat**: 输出可能很大（100-500行）
3. **systeminfo**: 输出固定（约50行）
4. **ipconfig**: 输出中等（20-50行）

## 故障排除

### 常见问题
1. **命令找不到**: 确保系统路径包含 System32 目录
2. **权限不足**: 以管理员身份运行命令
3. **中文乱码**: 确保输出编码正确设置为 GBK
4. **输出截断**: 增加执行缓冲区大小

### 错误代码
- **0**: 成功
- **1**: 一般错误
- **2**: 语法错误
- **9009**: 程序不是内部或外部命令，也不是可运行的程序
