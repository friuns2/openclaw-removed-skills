## 你将得到什么

- 7天可执行冲刺计划（每天练什么、练多久、如何验收）
- 100分岗位就绪度评分（是否可投递 + 风险项 + 两周提升路线）
- 一对一模拟面试（逐题提问、即时反馈、总结报告）
- 高分话术改写（30秒版 + 2分钟版 + 追问应对）
- 历史记录与能力雷达图（看见进步，持续优化）

## 3个真实使用场景

### 1) 不会规划：7天冲刺计划

**你说：**

> 我两周后面前端，给我一个7天冲刺计划

**你会得到：**

- Day1-Day7 每日任务
- 题型配比（问答/编程/系统设计/行为）
- 每天验收标准
- 是否需要升降难度的建议

### 2) 不知道能不能面：岗位就绪度评分

**你说：**

> 我现在能去面中级后端吗？

**你会得到：**

- Job Readiness Score（0-100）
- 五维评分（技术/编程/设计/行为/表达）
- Top 风险项
- 两周可执行提升路径

### 3) 知道但不会说：高分话术改写

**你说：**

> 帮我把这个答案改成高分表达

**你会得到：**

- 原答案问题点（2-3条）
- 30秒高分版（快速回答）
- 2分钟高分版（深入回答）
- 2个高频追问与应对

1. 安装 skill
2. 复制一句推荐提示词

推荐起手句：

- 3-5年：`我有4年后端经验，做一次岗位就绪度评估`
- 临近面试：`明天面试，先来一轮高压模拟面试`

## 核心能力

### 难度与题量智能匹配

- 难度：初级 / 中级 / 高级
- 题量：5 / 10 / 15 / 20 题
- 自动从工作年限与描述推断，也支持手动指定

### 五种题型覆盖真实面试

- 问答型：技术原理和深度
- 编程题：LeetCode 风格 + 复杂度要求
- 系统设计：架构、扩展性、容错
- 案例分析：业务分析与方案取舍
- 行为面试：STAR 结构化回答

### 双模式训练

- 文档模式：一次性生成完整练习文档
- 模拟面试模式：逐题提问 + 即时反馈 + 面试总结

### 历史追踪与能力分析

- 自动保存练习记录
- 查看趋势变化与阶段里程碑
- 生成能力雷达图
- 基于历史表现给个性化推荐

### 手动安装

1. 克隆或下载本项目
2. 将整个目录复制到你的 skills 目录：

```text

~/.claude/skills/ai-interview-coach/
```

3. 重启 AI 工具或刷新缓存
