# 蔚来 · NIO

---

## 发展历程

| 时期 | 关键事件 |
|:----:|--------|
| **2014 年** | 李斌在上海创立蔚来 |
| **2016 年** | 发布 EP9——刷新纽北圈速记录的电动超跑 |
| **2018 年** | 纽交所上市 |
| **2019 年** | 推出换电模式——3 分钟完成电池更换 |
| **2023 年** | 发布手机 + 芯片自研计划——生态扩展 |
| **2024 年** | 与长安/吉利/奇瑞达成换电合作 |

---

## 核心特点

高端电动车（ET7/ET5/ES6/EC7 等）+ 换电网络 + 用户社区（NIO House/Life）

## 核心优势

换电模式全球独有 + 用户社区运营（NIO Day 等）+ 高端品牌定位
