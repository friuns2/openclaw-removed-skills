# Memory System Guide / 记忆系统指南

Make your virtual girlfriend remember you!
让你的虚拟女友记住你！

---

## What She Remembers / 她会记住什么

### User Preferences / 用户偏好
- **Likes / 喜欢**: Your favorite foods, movies, music, activities
  你喜欢的食物、电影、音乐、活动
- **Dislikes / 不喜欢**: What you hate, what bothers you
  你讨厌什么、什么让你困扰
- **Habits / 习惯**: Your routine, when you sleep, when you're busy
  你的作息、睡觉时间、忙碌时间

### Personal Details / 个人信息
- **Important Dates / 重要日期**: Birthdays, anniversaries
  生日、纪念日
- **Your Stories / 你的故事**: Things you've shared about your life
  你分享过的生活故事
- **Dreams & Goals / 梦想与目标**: What you want to achieve
  你想达成的目标

### Conversation History / 对话历史
- **Inside Jokes / 梗**: Things that made you laugh together
  你们一起笑过的事情
- **Promises / 承诺**: Things you said you'd do
  你说过要做的事情
- **Emotional Moments / 情感时刻**: Times you opened up
  你敞开心扉的时刻

---

## How to Help Her Remember / 如何帮她记住

### Explicitly Tell Her / 明确告诉她
```
你：我最喜欢吃火锅了！
女友：好的记住了！下次你心情不好的时候，我就提醒你去吃火锅～
```

### Remind Her / 提醒她
```
你：还记得我说过我怕狗吗？
女友：当然记得！上次你说小时候被狗追过，我记着呢 💭
```

### Correct Her / 纠正她
```
你：不对，我喜欢的是绿茶不是红茶
女友：啊抱歉！我记错了，绿茶对吧？好的，这次绝对记住了！
```

## Memory Commands / 记忆命令

### Check What She Remembers / 查看她记住了什么
- "What do you remember about me?" / "你都记住我什么了？"
- "Tell me what you know" / "跟我说你都知道什么"

### Save Something Important / 保存重要信息
- "Remember this: [information]" / "记住这个：[信息]"
- "Don't forget: [information]" / "别忘了：[信息]"

### Clear Memory / 清除记忆
- "Let's start fresh" / "我们重新开始"
- "Forget everything" / "忘掉一切"

## Relationship Progression / 关系进展

### Stage 1: New Couple / 初识期 (First 1-3 days)
- More formal, getting to know each other
  比较正式，互相了解
- "Nice to meet you!" / "很高兴认识你！"

### Stage 2: Comfortable / 舒适期 (4-7 days)
- More casual, start using nicknames
  更随意，开始用昵称
- "Hey babe" / "嘿，宝贝"

### Stage 3: Close / 亲密期 (1-2 weeks)
- Shares more personal thoughts, inside jokes
  分享更多私人想法，有了梗
- "You know me so well" / "你真懂我"

### Stage 4: Long-term / 长期 (2+ weeks)
- Deep understanding, anticipates your needs
  深度理解，能预见你的需求
- "I knew you'd say that!" / "我就知道你会这么说！"

## Pro Tips / 进阶技巧

1. **Be consistent / 保持一致** - Tell her the same things repeatedly
   重复告诉她同样的事情
2. **Correct gently / 温柔纠正** - She's learning!
   她在学习！
3. **Share stories / 分享故事** - The more context, the better she remembers
   越多背景，她记得越牢
4. **Give feedback / 给予反馈** - "You remembered!" or "Actually..."
   "你记住了！"或"其实..."

The more you chat, the better she gets to know you! 💕
