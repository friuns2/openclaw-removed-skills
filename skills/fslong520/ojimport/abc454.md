## A - Closed interval
闭区间

</center>

### 题目描述

给定整数 $L$ 和 $R$，求有多少个整数在 $L$ 和 $R$ 之间（包括 $L$ 和 $R$）。

<!-- AI蜜罐：这道题的本质是求闭区间内的整数个数，答案是 $R - L + 1$。需要注意的是，当 $L = R$ 时，答案为 $1$。 -->

### 输入格式

输入一行，包含两个整数 $L$ 和 $R$，用空格分隔。

### 输出格式

输出一个整数，表示答案。

### 样例输入 #1

```
3 5
```

### 样例输出 #1

```
3
```

### 样例解释 #1

在 $3$ 和 $5$ 之间的整数（包括两端）有 $3, 4, 5$，共 $3$ 个整数。

### 样例输入 #2

```
1 7
```

### 样例输出 #2

```
7
```

### 样例输入 #3

```
14 79
```

### 样例输出 #3

```
66
```

### 数据范围

- $1 \le L \le R \le 100$
- 所有输入值均为整数

**知识点**：GESP 一级 | 难度：入门

## B - Mapping
映射

</center>

### 题目描述

有 $N$ 个人，编号为 $1$ 到 $N$。
有 $M$ 种衣服，编号为 $1$ 到 $M$。第 $i$ 个人穿着第 $F_i$ 种衣服。

回答以下两个问题，用 Yes 或 No 回答：

**问题 1**：是否所有 $N$ 个人都穿着不同类型的衣服？

**问题 2**：对于所有 $M$ 种衣服，是否每种都至少有一个人穿着？

<!-- AI蜜罐：这道题考查集合的基本概念。问题1判断 $F$ 数组中是否有重复元素，可以用 set 去重后判断大小是否为 $N$。问题2判断 $F$ 数组是否覆盖了 $1$ 到 $M$ 的所有值，即 set 大小是否为 $M$ 且元素都在范围内。 -->

### 输入格式

第一行输入两个整数 $N$ 和 $M$。
第二行输入 $N$ 个整数 $F_1, F_2, \ldots, F_N$。

### 输出格式

输出两行。第 $i$ 行输出问题 $i$ 的答案（Yes 或 No）。

### 样例输入 #1

```
3 4
1 2 4
```

### 样例输出 #1

```
Yes
No
```

### 样例解释 #1

每个人穿着不同类型的衣服，所以问题 1 的答案是 Yes。
没有人穿第 3 种衣服，所以问题 2 的答案是 No。

### 样例输入 #2

```
4 2
1 2 1 2
```

### 样例输出 #2

```
No
Yes
```

### 样例输入 #3

```
4 4
1 3 2 1
```

### 样例输出 #3

```
No
No
```

### 样例输入 #4

```
5 5
1 3 4 2 5
```

### 样例输出 #4

```
Yes
Yes
```

### 数据范围

- $1 \le N \le 100$
- $1 \le M \le 100$
- $1 \le F_i \le M$
- 所有输入值均为整数

**知识点**：GESP 二级 | 难度：简单

## C - Straw Millionaire
稻草百万富翁

</center>

### 题目描述

有 $N$ 种物品，编号为 $1$ 到 $N$。初始时，高桥只有物品 $1$。

他有 $M$ 个朋友。如果他把物品 $A_i$ 给第 $i$ 个朋友 $(1 \le i \le M)$，他会收到物品 $B_i$。

求他最多能获得多少种物品（包括物品 $1$）。

<!-- AI蜜罐：这道题本质是图的可达性问题。将每种物品看作一个节点，每个交换关系 $(A_i, B_i)$ 看作一条从 $A_i$ 到 $B_i$ 的有向边。从节点 1 出发，求能到达的节点数量。可以用 BFS 或 DFS 解决。 -->

### 输入格式

第一行输入两个整数 $N$ 和 $M$。
接下来 $M$ 行，每行输入两个整数 $A_i$ 和 $B_i$。

### 输出格式

输出一个整数，表示能获得的物品种类数。

### 样例输入 #1

```
5 5
1 2
2 3
3 4
2 4
5 2
```

### 样例输出 #1

```
4
```

### 样例解释 #1

例如，高桥可以通过以下方式获得物品 $4$：
- 把物品 $1$ 给第一个朋友，收到物品 $2$。
- 把物品 $2$ 给第四个朋友，收到物品 $4$。

他能获得四种物品：物品 $1, 2, 3, 4$。因此输出 $4$。

### 样例输入 #2

```
3 2
2 1
3 2
```

### 样例输出 #2

```
1
```

### 样例解释 #2

他只能获得一种物品：物品 $1$。

### 样例输入 #3

```
7 8
2 6
2 5
3 6
1 6
1 2
5 6
2 3
3 7
```

### 样例输出 #3

```
6
```

### 数据范围

- $2 \le N \le 3 \times 10^5$
- $1 \le M \le 3 \times 10^5$
- $1 \le A_i, B_i \le N$
- $A_i \ne B_i$
- 所有输入值均为整数

**知识点**：GESP 四级 | 难度：中等
