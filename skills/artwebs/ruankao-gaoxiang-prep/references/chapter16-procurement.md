# 第16章 采购管理

## 背诵重点

1. 采购管理三个过程：规划采购管理、实施采购、控制采购
2. 协议类型：合同、服务水平协议（SLA）、谅解备忘录（MOA）、协议备忘录（MOA）或订购单
3. 合同三种类型（按范围）：总承包合同（买方将项目的全过程作为一个整体发包给同一个卖方）、单项承包合同（一个卖方只承包某一项或某几项内容）、分包合同（经合同约定和买方认可，卖方将其承包项目的某一部分或某几部分再发包给具有相应资质条件的分包方）
4. 合同六种类型（按付费方式）：总价合同（固定总价合同FFP、总价加激励费用合同FPIF、总价加经济价格调整合同FPEPA）、成本补偿合同（成本加固定费用合同CPFF、成本加激励费用合同CPIF、成本加奖励费用合同CPAF）、工料合同（T&M）、订购单
5. 合同内容11项：①项目名称②标的的内容和范围③项目的质量要求④项目的计划、进度、地点、交付物和方式⑤项目建设过程中的各种期限⑥技术情报和资料的保密⑦风险责任的承担⑧技术成果的归属⑨验收的标准和方法⑩价款、报酬及其支付方式⑪违约金或者损失赔偿的计算方法
6. 合同管理包括：合同签订管理、合同履行管理、合同变更管理、合同档案管理、合同违约索赔管理
7. 总价合同特点：工作范围很明确，项目设计已具备详细细节，卖方承担成本风险；适用范围清楚且风险最小
8. 成本补偿合同特点：工作范围尚不清楚，买方承担成本风险；适用范围不清楚
9. 工料合同特点：工作性质清楚、工作范围比较明确但具体工作量无法确定，买方承担工作量变动的风险，卖方承担单价风险，兼具成本补偿合同和总价合同的某些特点
10. 合同风险承担：成本补偿合同买方承担，工料合同双方承担，总价合同乙方承担

## 补充说明

**合同三种类型（按范围）对比：**
- 总承包合同：买方将项目的全过程作为一个整体发包给同一个卖方；一般适用于经验丰富、技术实力雄厚且组织管理协调能力强的卖方
- 单项承包合同：一个卖方只承包某一项或某几项内容，买方分别与不同卖方订立合同；有利于吸引更多卖方参与投标
- 分包合同：经合同约定和买方认可，卖方将其承包项目的某一部分或某几部分再发包给具有相应资质条件的分包方；必须同时满足5个条件：①经过买方认可②分包的部分必须是项目非主体工作③只能分包部分项目而不能转包整个项目④分包方必须具备相应的资质条件⑤分包方不能再次分包

**合同六种类型（按付费方式）对比：**
1. 总价合同（固定总价合同FFP）：价格在开始就确定，不允许改变（除非工作范围发生变更）；卖方承担成本超支风险
2. 总价加激励费用合同（FPIF）：允许一定的绩效偏离，并对实现既定目标给与相关的财务奖励；设置价格上限
3. 总价加经济价格调整合同（FPEPA）：适用于卖方履约期将跨越几年、将以不同货币支付；允许根据条件变化（如通货膨胀）对合同价格进行调整
4. 成本加固定费用合同（CPFF）：为卖方报销履行合同工作所发生的一切可列支成本，并向卖方支付固定费用
5. 成本加激励费用合同（CPIF）：为卖方报销履行合同工作所发生的一切可列支成本，并在卖方达到合同规定的绩效目标时向卖方支付预先确定的激励费用
6. 工料合同（T&M）：按项目工作所花费的实际工时数和材料数，按事先确定的单位工时费用标准和单位材料费用标准进行付款

**合同类型选择：**
- 范围明确、设计详细→总价合同
- 范围不清→成本补偿合同或工料合同
- 工作性质清楚、范围不太明确、工作不复杂、需要快速签订合同→工料合同

**成本加激励费用合同（CPIF）计算：**
- 如果实际成本大于目标成本，卖方可以得到的付款总数为：目标成本+目标费用+买方应负担的成本超支
- 如果实际成本小于目标成本，卖方可以得到的付款总数为：目标成本+目标费用-买方应享受的成本节约

**总价加激励费用合同（FPIF）计算：**
- 利润=目标利润-(实际成本-目标成本)×卖方比例
- 如果实际成本高于较高价，买方只支付较高价

**成本加固定费用合同（CPFF）计算：**
- 买方支付总价=实际成本+固定费用

**成本加成本百分比合同（CPPC）计算：**
- 买方支付总价=实际成本+实际成本×费用百分比

**合同风险承担：**
- 范围清楚→总价合同，风险最小，乙方承担
- 范围不清楚→成本补偿合同，成本实报实销，甲方承担
- 双方分担→工料合同

**合同管理内容：**
1. 合同签订管理
2. 合同履行管理
3. 合同变更管理
4. 合同档案管理
5. 合同违约索赔管理

**采购管理裁剪考虑因素：**
- 采购的复杂性
- 物理地点
- 治理和法规环境
- 承包商的可用性

**敏捷与适应方法：**
- 在敏捷或适应型环境中，可能需要与特定卖方进行协作来扩充团队
- 可以通过主体协议（如主要服务协议MSA）来管理整体协作关系
- 针对某些可交付成果采用敏捷或适应型方法，而对其他部分则采用更稳定的方法

**买方和卖方关系：**
- 卖方：承包商、供货商、服务提供商或供应商
- 买方：最终产品的所有人、分包商、收购机构、服务需求者或购买方
