# 第23章 组织通用管理

## 背诵重点

1. 人力资源管理目标：建立员工招聘和选择体系、充分挖掘每个员工的潜能、留住那些通过自己的工作绩效助力组织实现目标的员工、确保组织遵守人力资源方面的法律、法规、政策和标准
2. 人力资源内容：规划、招聘、维护、提升、评价
3. 工作分析目标：明确所要完成的任务以及完成这些任务所需要的人员能力特征，工作分析将每项工作所包含的任务、责任和任职资格用正式的文件明确下来
4. 工作分析流程：明确工作分析范围→确定工作分析方法→工作信息收集和分析→向组织报告结果→定期检查工作分析情况
5. 工作分析方法：定性（定性工作实践法、直接观察法、面谈法、问卷法和典型事例法）、定量（岗位分析问卷法、管理岗位描述问卷法和功能性工作分析法）
6. 工作设计内容：工作内容设计（工作的广度、工作的深度、工作的完整性、工作的自主性、工作的反馈性）、工作职责设计（工作的责任、权利、方法以及工作中的相互沟通）
7. 薪酬等级结构构成要素：薪酬等级数、目标薪酬、薪酬级差、薪酬幅度、薪酬重叠情况
8. 薪酬体系：直接薪酬（基本薪酬、绩效加酬、激励薪酬、延期支付）、间接薪酬（组织在这些方面为员工提供的薪酬除受到政府有关法律法规的限制以外，还受到市场竞争的影响）
9. 四类不同人员激励方式：操作人员（计件制、完全计件制、部分计件制）、销售人员（佣金制/底薪制，含佣金、固定底薪、混合制）、专业技术人员（加薪+非金钱奖励）、高级管理人员（薪酬+长期激励）
10. 薪酬调整：按调整性质分类（主动型调整vs被动型调整）、按调整内容分类（激励性调整、生活指数性调整、年资性调整、效益性调整）

## 补充说明

**组织战略类型：**
1. 增长型战略：寻求扩大组织的经营业务
2. 稳定型战略：保持组织现有业务结构
3. 紧缩型战略：减少组织经营业务
4. 组合型战略：同时实行上述两种或两种以上战略

**工作分析流程四个步骤：**
1. 明确工作分析范围：确立工作分析的目的、确定工作分析的对象
2. 确定工作分析方法：确定所需信息的类型、识别工作信息的来源、明确工作分析的具体步骤
3. 工作信息收集和分析：收集工作信息、分析所收集的信息、向组织报告结果、定期检查工作分析情况
4. 评价工作分析方法：以收益、成本、合规性和合法性等为标准评价工作分析的结果

**工作分析方法：**
1. 定性方法：定性工作实践法、直接观察法、面谈法、问卷法和典型事例法
2. 定量方法：岗位分析问卷法、管理岗位描述问卷法和功能性工作分析法

**工作设计原则：**
1. 工作的广度：应尽量使工作多样化，使员工在完成任务的过程中能进行不同的活动，保持对工作的兴趣
2. 工作的深度：设计的工作应具有从易到难的一定层次，对员工工作的技能提出不同程度的要求，从而增强工作的挑战性，激发员工的创新力和克服困难的能力
3. 工作的完整性：保证工作的完整性使员工有成就感，即使是流水作业中的一个简单程序，也要是全过程，让员工见到自己的工作成果，感受到自己工作的意义
4. 工作的自主性：适当的自主权利能增强员工的工作责任感，使员工感到自己受到信任和重视，认识到自己工作的重要性，增强工作的责任心，提高工作的热情
5. 工作的反馈性：工作的反馈性包括两方面的信息：一是同事及上级对自己工作意见的反馈，二是对工作本身的反馈，如工作的质量、数量、效率等

**岗位设计内容：**
1. 工作内容设计：是岗位设计的重点，一般包括工作的广度、工作的深度、工作的完整性、工作的自主性、工作的反馈性5个方面
2. 工作职责设计：包括工作的责任、权利、方法以及工作中的相互沟通等方面

**岗位设计方法：**
科学管理方法、人际关系方法、工作特征模型、高绩效工作系统

**人力资源计划三个部分：**
1. 供给报表
2. 需求报表
3. 人力报表

**实施工作评价方法：**
工作排序法、因素比较法、工作分类法、点数法和海氏系统法

**薪酬等级结构五个构成要素：**
1. 薪酬等级数：划分的等级数量（太少损害公平性，太多造成混乱）
2. 目标薪酬：每级职位的目标薪酬（通常称为中点、基准点等）
3. 薪酬级差：薪酬等级中相邻两个等级的目标薪酬之间的差额
4. 薪酬幅度：每级职位薪酬的范围幅度（即以目标薪酬为中点，薪酬最低点与最高点之间的差额）
5. 薪酬重叠情况：相邻级别薪酬区间重叠程度（避免薪酬增长天花板）

**四类不同人员激励方式：**
1. 操作人员：计件制（完全计件制、部分计件制），直接有效但可能忽视质量
2. 销售人员：佣金制/底薪制（含佣金、固定底薪、混合制），激励性强但可能过于短视
3. 专业技术人员：加薪+非金钱奖励（年资加薪、设备改善、支持性管理），符合专业人员特点
4. 高级管理人员：薪酬+长期激励（基本薪酬、短期奖励或奖金计划、长期激励计划、正常雇员福利、特殊福利或津贴），全面激励，兼顾长短期

**薪酬调整分类：**
1. 按调整性质分类：主动型调整（组织主动行为）、被动型调整（外部因素压力）
2. 按调整内容分类：
   - 激励性调整：激励优异绩效（个别员工）
   - 生活指数性调整：弥补通胀损失（全体员工普调）
   - 年资性调整：根据工龄经验（全员性常规调整）
   - 效益性调整：根据组织效益变化（全体员工分享/共担）

**流程管理五个环节：**
1. 流程基础
2. 流程规划（岗位职级开始或业务模型开始）
3. 流程执行
4. 流程评价
5. 流程持续改进

**流程规划参考方法：**
1. 岗位职级开始（从下到上）：优点：工作分析细致透彻，不容易遗漏，因整个过程中流程管理部门起主导作用，对被访谈人的流程管理方面的专业知识、技能和经验要求不高，各级流程干系人充分参与，工作成果容易被接受，应用较容易
   缺点：工作量比较大，工作质量容易受访谈人的工作经验及描述工作能力影响
2. 业务模型开始（从上到下）：优点：工作量相对比较小，流程管理部门对整个工作控制力度大，工作进度和风险易于控制
   缺点：因为没有对工作进行详细的分析，工作容易出现遗漏，对参与人员的流程规划专业能力要求较高，由于各级流程干系人未充分参与，工作成果可能不被认可

**流程优化需求三种类型：**
1. 问题导向：解决组织分散的问题
2. 绩效导向：提升业务增长和盈利
3. 变革导向：应对内外部环境变化

**显性知识和隐性知识：**
显性知识特征：非陈述性、可共享性、静态存在性、客观存在性
隐性知识特征：非陈述性、个体性、实践性

**知识管理三条原则：**
积累原则、共享原则、交流原则

**知识管理过程：**
1. 知识获取
2. 知识沉淀
3. 知识共享
4. 知识应用
5. 知识创新

**显性知识和隐性知识：**
显性知识：以文字、图表、公式等形式存在的知识，容易被记录、传递和存储
隐性知识：存在于人的大脑中的经验、直觉、技能等，难以被记录、传递和存储，通过实践、观察、交流等方式获取

**市场营销4P：**
1. 产品
2. 定价
3. 渠道
4. 促销

**营销环境分析：**
政治环境、经济环境、社会环境、技术环境

**营销分析：**
市场分析、客户分析、竞争对手分析
