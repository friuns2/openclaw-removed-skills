# 第3章 项目立项管理

## 背诵重点

1. 甲方立项流程：需求调研→编写项目申请书→可行性研究（机会、初步、详细）→项目论证→项目评估→评审获得批准→发布招标文件
2. 乙方立项流程：看到招标文件→进行项目识别→可行性研究（机会、初步、详细）→项目论证→项目评估→参加投标→中标拿到项目→签订合同
3. 项目建议书核心内容：项目的必要性、项目的市场预测、项目预期成果的市场预测、项目建设必需的条件【1必要性、2预测、1条件】
4. 可行性研究内容：技术可行性分析、经济可行性分析、社会效益可行性分析、运行环境可行性分析、其他方面（法律可行性、政策可行性、合同责任、知识产权）
5. 技术可行性分析考虑：进行项目开发的风险、人力资源的有效性、技术能力的可能性、物资（产品）的可用性【风险、人力、技术、物资】
6. 经济可行性分析包括：支出分析、收益分析、投资回报分析、敏感性分析
7. 社会效益可行性：组织内部（品牌效益、技术创新效益、竞争力效益、管理提升效益、人员提升收益）、组织外部（公共效益、环境效益、文化效益、社会责任感效益、其他收益）【品技竞管人，公环文责其】
8. 项目总成本四大类：研发成本、行政管理费、销售与分销费用、财务费用和折旧。前三类成本总和称为经营成本
9. 项目评估依据：①项目建议书及其批准文件；②项目可行性研究报告；③报送组织的申请报告及主管部门的初审意见；④项目关键建设条件和工程等的协议文件；⑤必需的其他文件和资料【可以初见】
10. 项目评估工作程序：成立评估小组→开展调查研究→分析与评估→编写、讨论、修改评估报告→召开专家论证会→评估报告定稿并发布

## 补充说明

**可行性研究三个层次：**
- 机会可行性研究：快速判断项目是否值得进一步研究
- 初步可行性研究：对项目进行初步评估，可以形成初步可行性研究报告
- 详细可行性研究：不可省略，是项目决策的最终依据

**详细可行性研究的方法：**
经济评价法、市场预测法、投资估算法、增量净效益法（有无比较法）

**初步可研、详细可研、辅助研究的内容对比：**
| 初步可研 | 详细可研 | 辅助研究 |
|---------|---------|---------|
| 需求与市场预测 | 市场需求预测 | 对要设计开发的产品进行的市场研究 |
| 设备与资源投入分析 | 技术与设备选择 | 设备选择研究 |
| 项目投资与成本估算 | 投资、成本估算与资金筹措 | 规模的经济性研究 |
| 空间布局 | 网络物理布局设计 | 网络物理布局设计 |
| 部件和投入的选择供应 | 配件和投入物资的研究 | 配件和投入物资的研究 |
| 项目设计 | 信息系统架构及技术方案的确定 | 试验室和中间工厂的试验 |
| 项目进度安排 | 经济评价及综合分析 | 经济评价及综合分析 |

**经济分析关键指标：**
- 净现值（NPV）：未来现金流的现值减去初始投资，NPV>0则项目可行
- 内部收益率（IRR）：使项目净现值为零的折现率，IRR>基准收益率则项目可行
- 投资回收期：收回初始投资所需的时间，回收期越短项目越优
- 盈亏平衡点：收入等于成本时的产量或销售额
- 敏感性分析：当关键因素（设备、软件配置、处理速度等）变化时，对支出和收益产生影响的估计

**项目评估报告内容大纲：**
项目概况、详细评估意见、总结和建议
