# 第21章 运筹学计算

## 背诵重点

1. 复利法公式：F=P×（1+i）n其中F为终值P为现值i为年利率n为年份
2. 单利法：利息=本金×利率×年限
3. 复利法：利息=本金×[（1+i）n-1]
4. 净现值：将未来某一时点的资金金额换算成现在时点的等值金额称为折现或贴现
5. 静态投资回收期：用多长时间能把项目投资收回来计算公式为（T-1）+第（T-1）年累计现金流量绝对值/第T年现金流量
6. 动态投资回收期：考虑货币时间价值的投资回收期计算公式为（T-1）+第（T-1）年累计折现值/第T年折现值
7. 投资收益率：ROI=EBIT/TI×100%其中EBIT为达产后正常年份的年息税前利润TI为投资总额
8. 伏格尔法口诀：算差→找最大→分配→划掉→重复
9. 匈牙利法步骤：行化简（每行减去最小值）→列化简（每列减去最小值）→试分配（寻找独立零元素）→验证分配方案
10. 运筹学常考题型：最短和最长路径问题线性规划问题最小生成树匈牙利法后悔值问题运输问题解决资源分配

## 补充说明

**一、工程经济学**

**资金的时间价值：**
1. 资金的时间价值是指不同时间发生的等额资金在价值上的差别
2. 把资金存入银行经过一段时间后也会产生增值这就是利息
3. 资金的时间价值公式：F=P×（1+i）n

**利息计算方法：**

**1. 单利法：**
- 利息=本金×利率×年限
- 本利和=本金+利息=本金×（1+利率×年限）

**2. 复利法：**
- 利息=本金×[（1+i）n-1]
- 本利和=本金×（1+i）n
- 每一计息周期的利息都要并入下一期的本金再计利息

**净现值：**
- 未来某一时点的资金金额换算成现在时点的等值金额的过程称为折现或贴现
- 与现值等价的将来某时点的资金价值称为终值或未来值
- 现值是指资金现在的瞬时价值

**投资回收期：**
- 静态投资回收期：用多长时间能把项目投资收回来计算公式为（T-1）+第（T-1）年累计现金流量绝对值/第T年现金流量
- 动态投资回收期：考虑货币时间价值的投资回收期计算公式为（T-1）+第（T-1）年累计折现值/第T年折现值
- 投资回收期的优点是简单易懂缺点是不考虑货币的时间价值也不考虑投资全部回收之后的项目产品运营收入

**投资收益率法：**
- 投资收益率ROI=EBIT/TI×100%
- EBIT为达产后正常年份的年息税前利润
- TI为投资总额
- 当ROI≥基准投资收益率Rb时项目可行

**起算点记忆技巧：**
- 如果是2006年2007年2008年这样给出的则默认2006年为第0年2007年为第1年则默认从0开始
- 如果是1234或者第1年第2年这样给出的则默认从1开始

**二、线性规划**

**线性规划问题：**
- 目标是在一组线性约束条件下寻找线性目标函数的最大值或最小值
- 常见题型：最大利润的生产安排最小成本的资源分配等

**解题步骤：**
1. 根据题意列出目标函数（如最大利润或最小成本）
2. 根据资源约束列出约束条件
3. 解方程组或用代入法求解
4. 比较各方案的结果选择最优解

**应用场景：**
1. 产品生产优化
2. 资源分配优化
3. 成本最小化
4. 利润最大化

**三、伏格尔法（运输问题）**

**运输问题定义：**
多个仓库向多个工地或需求点配送资源已知各路线运费求总运费最小的调配方案

**伏格尔法核心记忆：**
伏格尔法=每次优先满足"代价差最大"的行或列本质是机会成本最大优先

**伏格尔法详细步骤：**

**步骤1：建表**
将供应量、需求量、运费填入运输表

**步骤2：算罚数（差额）**
每行：找该行最小运费和次小运费相减得到行罚数
每列：找该列最小运费和次小运费相减得到列罚数
罚数含义：若错过最优路线需多付的"代价"

**步骤3：找最大罚数**
从所有行列罚数中找最大值选该行或列优先分配

**步骤4：分配**
在最大罚数对应的行列中选运费最小的格分配
分配量=该行供应量该列需求量

**步骤5：划掉已满足的行或列**
若某行供应量用完→划掉该行
若某列需求量满足→划掉该列
若同时满足→划掉一个另一个保留（供应量或需求量记为0）

**步骤6：重复**
重新计算剩余行列的罚数重复第二至五步直到所有供应量和需求量全部分配完毕

**步骤7：计算总运费**
将所有分配格的运费×分配量求和

**伏格尔法易错点：**
1. 罚数算错一定是最小费用与次小费用之差不是其他
2. 并列罚数不知如何选任选其一结果相差不大不影响得分
3. 划掉行列后忘记更新罚数每轮必须重新计算剩余行列的罚数
4. 同时耗尽行和列只划掉一个另一个写0继续参与下一轮
5. 供需总量不等先检查：供应总量≠需求总量时需补虚拟行列（费用填0）

**解题模板（考场速用）：**
建表→算每行列罚数（次小-最小）→找最大罚数所在行列→在该行列选最小费用格分配（取min值）→划掉已满足的行或列→重复②~⑤直至分配完毕→求和：Σ（费用×分配量）

**四、匈牙利法（指派问题）**

**指派问题定义：**
有n项任务恰好有n个人可以分别去完成其中任何一项由于任务的性质和每个人的技术专长各不相同因此各人去完成不同任务的效率也不一样问题是指派哪个人去完成哪项任务才能使总的效率最高

**匈牙利法详细步骤：**

**步骤1：行化简**
对每一行减去该行的最小值
得到至少包含一个零的新矩阵
目的是在每行中产生至少一个零元素

**步骤2：列化简**
对每一列减去该列的最小值
继续在每列中产生零元素
目的是在每列中产生至少一个零元素

**步骤3：试分配（寻找独立零元素）**
寻找矩阵中的零元素
尝试进行任务分配
优先选择所在行或列中零元素最少的位置
确保每个人和每项任务都恰好分配一次

**步骤4：验证分配方案**
检查是否每个人和每项任务都恰好分配一次
如果分配不完整需要重新调整
如果分配完整则计算总成本或时间

**匈牙利法要点：**
1. 一般先行后列当然先列后行也行
2. 如果一个算不出来可以换个思路
3. 优先选择所在行或列中零元素最少的位置
4. 确保每个人和每项任务都恰好分配一次
5. 对于非标准指派问题需要综合考虑各种可能的分配方案

**指派问题的类型：**
1. 标准指派问题：任务数=人数每人完成一项任务
2. 非标准指派问题：
   - 任务数>人数：有人需要完成多项任务
   - 任务数<人数：有人无法分配到任务
   - 目标是最小化总成本或时间

**五、最短路径问题**

**最短路径定义：**
在图论中寻找从一个节点到另一个节点的最短路径通常使用Dijkstra算法或观察法

**解题方法：**
1. 观察法：对于小规模问题可以通过观察直接找到最短路径
2. Dijkstra算法：对于大规模问题可以使用Dijkstra算法系统求解
3. 动态规划：对于特定类型的问题可以使用动态规划方法

**解题步骤：**
1. 识别起始节点和目标节点
2. 列出从起始节点出发的所有可能路径
3. 计算每条路径的总长度
4. 选择长度最短的路径作为最优解

**应用场景：**
1. 物流配送优化
2. 网络路由选择
3. 交通路径规划
4. 项目进度优化

**六、资源分配问题**

**资源分配定义：**
在有限资源约束下将资源分配给多个项目或方向以获得最大收益或利润

**解题方法：**
1. 穷举法：对于小规模问题可以列举所有可能的分配方案选择最优解
2. 动态规划：对于大规模问题可以使用动态规划方法
3. 贪心算法：对于特定类型的问题可以使用贪心算法

**解题步骤：**
1. 识别可分配的资源总量
2. 列出各个项目或方向的收益表
3. 根据资源约束列出所有可能的分配方案
4. 计算每种分配方案的总收益
5. 选择收益最大的方案作为最优解

**应用场景：**
1. 资金投资分配
2. 人力资源配置
3. 生产资源优化
4. 研发经费分配

**七、最小生成树问题**

**最小生成树定义：**
在给定连通图中找到总权值最小的生成树使得图中所有节点都连通且总权值最小

**破圈法求最小生成树：**
1. 在给定的赋权的连通图上任找一个圈
2. 在所找的圈中去掉一条权数最大的边（如果有两条或两条以上的边都是权数最大的边则任意去掉其中一条）
3. 如果所余下的图已不含圈则计算结束所余下的图即为最小生成树否则返回步骤1

**贪心算法求最小生成树：**
1. 将所有边按权值从小到大排序
2. 依次选择权值最小的边
3. 如果选择的边会形成环则跳过
4. 直到所有节点都连通

**解题步骤：**
1. 识别图中的所有节点和边
2. 使用破圈法或贪心算法求解
3. 计算最小生成树的总权值
4. 验证所有节点是否都连通

**应用场景：**
1. 网络线路规划
2. 通信网络设计
3. 管道铺设优化
4. 交通网络建设

**八、运筹学复习建议**

**复习重点：**
1. 运筹学常考的有最短和最长路径问题线性规划的问题最小生成树匈牙利法后悔值的问题运输问题解决资源分配
2. 其他类型的题目一般可以通过穷举法去解题如果这里实在有困难可以放弃处理
3. 本章节内容2025年2批次均考了案例分析因此以后的学习要当做案例分析的考点对待

**复习技巧：**
1. 常见题型1-2分+正确思路或代入法解题1-2分+运气
2. 思路比记住以往考题更重要
3. 优先考虑看能不能用代入法实在没思路看能不能用穷举法
4. 能用某种计算思路凑对答案选项的可以优先选
5. 注意：很多题的计算方法并不只有一种
6. 尽量得分实在不会穷举法没有时间的话就猜吧B或者C

**注意事项：**
1. 复习时在运筹学上花的时间要适可而止没有太多时间的重点掌握视频课里的题型即可
2. 一定要会举一反三历年真题及详细解析必须搞懂
3. 运筹学是一门专门的学科博大精深在考试中通常会考察5分且计算量很大
4. 如果花大量时间专门去复习它性价比不高
