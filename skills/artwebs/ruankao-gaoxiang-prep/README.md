## 🚀 快速开始

### 1. 导入到 OpenClaw

技能已安装在 `~/.openclaw/skills/ruankao-gaoxiang-prep/`，OpenClaw 会自动识别。

### 2. 设置每日推送（默认 09:00）

**方法一：通过 QQ 机器人设置**

在 QQ 中发送以下消息给机器人：

```
设置软考备考推送
```

机器人会自动创建每天 9:00 的定时推送任务。

**方法二：通过命令行设置**

```bash
# 获取用户的 openid
# (在 QQ 中与机器人对话，机器人会在日志中显示 openid)

# 创建定时任务（替换 {your-openid} 为实际的 openid）
cat <<EOF > /tmp/ruankao-cron.json
{
  "action": "add",
  "job": {
    "name": "软考高项每日推送",
    "schedule": {
      "kind": "cron",
      "expr": "0 9 * * *",
      "tz": "Asia/Shanghai"
    },
    "sessionTarget": "isolated",
    "wakeMode": "now",
    "payload": {
      "kind": "agentTurn",
      "message": "请执行以下任务：(1) 调用 scripts/daily_push.py 获取今天的推送内容 (2) 输出格式：'📚 软考高项每日复习 {日期}\\n\\n📖 今日重点（第X章）：\\n{10条重点内容}\\n\\n📝 今日英语单词：\\n{10个单词}\\n\\n🎯 历年真题精选（10题）：\\n{10道题目}\\n\\n💪 加油，坚持就是胜利！' (3) 不要回复HEARTBEAT_OK (4) 用emoji点缀让内容更生动",
      "deliver": true,
      "channel": "qqbot",
      "to": "55DE97983076AF00410248091457E495"
    }
  }
}
EOF

# 提交定时任务
openclaw cron create /tmp/ruankao-cron.json
```

### 3. 自定义推送时间

如果你想修改推送时间，可以编辑 cron 表达式：

| 推送时间 | Cron 表达式 |
|---------|-------------|
| 每天 8:00 | `"0 8 * * *"` |
| 每天 9:00 | `"0 9 * * *"` |
| 每天 12:00 | `"0 12 * * *"` |
| 每天 20:00 | `"0 20 * * *"` |

在 QQ 中发送以下消息修改时间：

```
设置软考备考推送，早上8点
```

### 推送逻辑

1. **日期轮询**：根据一年中的第几天（1-366）决定推送章节
2. **公式**：`章节号 = (天数 % 24) + 1`
3. **内容生成**：
   - 读取对应章节的 `references/chapterXX-xxx.md`
   - 提取前10条重点内容
   - 读取 `references/english-words.md` 获取10个单词
   - 读取 `references/exam-questions.md` 获取10道真题
   - 格式化输出

## 📝 常见问题

### Q: 如何修改推送时间？
A: 在 QQ 中发送「设置软考备考推送，{时间}」，例如：
- 「设置软考备考推送，早上8点」
- 「设置软考备考推送，晚上8点」

### Q: 如何取消每日推送？
A: 发送「取消软考备考推送」给机器人。

### Q: 推送内容包括哪些？
A: 每天推送包含：
- 10条章节重点内容
- 10个英语单词及音标
- 10道历年真题（含答案）

### Q: 章节是按什么顺序推送的？
A: 按日期轮询，一年366天对应24章循环。例如：
- 第1天 → 第1章
- 第2天 → 第2章
- ...
- 第24天 → 第24章
- 第25天 → 第1章（循环）

### Q: 如何查看特定章节的内容？
A: 在 QQ 中发送「{章节名}重点」，例如：
- 「第一章重点」
- 「成本管理重点」
- 「第11章重点」

### Q: 技能支持哪些平台？
A: 目前支持 QQ 机器人（qqbot）。
