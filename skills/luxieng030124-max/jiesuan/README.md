# AI智能结算助手 Pro

## 📦 Skill包说明

这是一个为OpenClaw设计的AI智能结算助手技能包，支持活动结算的自动化处理。

### 版本信息
- **版本号**: v1.1.0
- **发布日期**: 2026-04-21
- **Skill ID**: ai-settlement-pro

---

## ✨ 核心功能

1. **🏷️ 话题词精准识别**
   - 精准匹配视频标题中的话题词（#金铲铲）
   - 支持且/或逻辑关系
   - 自动区分相似话题词

2. **✅ 规则确认流程**
   - AI解析后返回格式化的规则理解
   - 支持多轮修正
   - 确认后再执行结算

3. **📊 多模式结算**
   - 达标瓜分模式
   - 排名赛模式
   - 混合不互斥模式
   - 权重分配模式

## 🚀 快速开始

### 安装方法

将整个 `ai-settlement-pro` 文件夹放入OpenClaw的技能目录：

```
~/.openclaw/skills/ai-settlement-pro/
```

或通过OpenClaw技能市场安装。

### 基本使用

```
请帮我结算活动：总奖金2万，播放量≥3万，携带话题#金铲铲的作者瓜分
```

---

## 🏷️ 话题词规则示例

```
# 单个话题词（OR逻辑）
"总奖金2万元，播放量≥3万，必须携带话题 #金铲铲 的作者等额瓜分"

# 多个话题词（AND逻辑）
"奖池5万元，播放量≥10万，同时携带话题 #金铲铲 和 #攻略 的作者瓜分"

# 多个话题词（OR逻辑）
"总奖金3万，作品≥3条，携带话题 #英雄联盟 或 #云顶之弈 的用户瓜分"
```

---

## 📋 数据要求

CSV/Excel文件需包含以下字段：

| 字段名 | 必需 | 说明 |
|--------|------|------|
| 作者ID | ✅ | 用户唯一标识 |
| 作者名称 | ✅ | 用户显示名称 |
| 视频标题 | ✅ | 用于话题词匹配 |
| 播放量 | 根据规则 | 结算依据字段 |
| 点赞数 | 根据规则 | 排名字段 |
