# 📰 微信文章抓取器 - 安全版

> 使用 Puppeteer + Chrome 本地提取微信公众号文章内容，支持全文提取和元信息获取。

---

## ⚡ 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 运行

```bash
node fetch-wechat-article.js https://mp.weixin.qq.com/s/xxx
```

---

## 📋 功能

- ✅ 微信公众号文章全文提取
- ✅ 元信息获取（标题、作者、时间）
- ✅ 本地运行，数据不外传
- ✅ JavaScript 渲染支持
- ✅ 自动保存为文本文件
- ✅ 错误处理和超时保护
