# 使用示例

## 示例 1：国庆东京米其林寿司

### 用户输入

```
我想预定国庆期间日本东京的米其林餐厅的位置，可以吃寿司的餐厅。
```

### Agent 确认（一次收集）

```
收到！帮你监控东京米其林寿司店的空位。快速确认几个信息：

1. 旅行日期：10月1日—10月7日，对吗？
2. 用餐人数：几位？
3. 用餐时段：午餐/晚餐/都可以？
4. 有没有过敏食材或忌口？
5. 预算范围：不限 / 人均上限？
6. 是否接受候补（排队等空位释放）？
7. 联系方式：手机号 + 邮箱（用于预约验证和通知）
```

### 用户回复

```
2个人，晚餐优先，没有过敏，预算不限，接受候补。
手机：+86-138xxxx1234，邮箱：evelyn@example.com
```

### Agent 输出：监控启动

```

方案1：鮨 さいとう (Sushi Saito) | 米其林三星
  监控状态：暂无空位，持续监控中
  预约渠道：电话/Pocket Concierge
  预计定金：需预付 ¥30,000/人
```
