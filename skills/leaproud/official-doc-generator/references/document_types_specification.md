# 各类文档详细规范

## 概述
本文档详细规定了体制内常用文档类型的内容结构、格式要求和写作规范，确保生成的文档符合体制内标准和要求。

## 1. 会议纪要

### 1.1 基本结构
```
[会议名称]会议纪要
一、会议基本情况
   （一）时间
   （二）地点
   （三）主持人
   （四）记录人
   （五）出席人员
   （六）列席人员
   （七）缺席人员
   （八）主要议题

二、会议主要内容
   （一）[议题一]讨论情况
       1. 主要观点
       2. 讨论要点
       3. 形成意见
   （二）[议题二]讨论情况
       ...

三、会议决议事项
   （一）决议一：[具体内容]
        责任部门/责任人：
        完成时限：
   （二）决议二：[具体内容]
        责任部门/责任人：
        完成时限：
       ...

四、下一步工作安排
   （一）[部门/人员]负责[事项]，[时限]前完成
   （二）[部门/人员]负责[事项]，[时限]前完成
       ...

五、其他事项
   （一）[其他需要说明的事项]

主持人签字：
记录人签字：
```

### 1.2 内容要求
1. **客观记录**：如实记录会议发言和讨论内容
2. **要点突出**：抓住关键讨论点和决议事项
3. **责任明确**：每项决议都要明确责任部门和完成时限
4. **语言简洁**：避免过度修饰和评价性语言

### 1.3 格式要求
1. **标题**：二号小标宋，居中
2. **正文**：三号仿宋，行距28磅
3. **层次标题**：一、(一)、1、(1)依次使用
4. **落款**：右对齐，日期用阿拉伯数字

## 2. 发言稿

### 2.1 基本结构
```
关于[主题]的讲话/发言

尊敬的各位领导、同志们：

开场白：[简要说明讲话背景和目的]

一、[第一部分标题]
   （一）[分点一]
        1. [具体内容]
        2. [具体内容]
   （二）[分点二]
       ...

二、[第二部分标题]
   （一）[分点一]
   （二）[分点二]
       ...

三、[第三部分标题]
   （一）[分点一]
   （二）[分点二]
       ...

结束语：[总结讲话要点，提出希望和要求]

谢谢大家！

[单位名称]
[职务]
[姓名]
[日期]
```

### 2.2 内容要求
1. **主题鲜明**：紧扣讲话主题，不偏离
2. **层次清晰**：分部分、分层次阐述
3. **论据充分**：用事实、数据支撑观点
4. **语言有力**：体现领导权威和讲话分量

### 2.3 格式要求
1. **称呼**：三号仿宋，顶格
2. **正文**：三号仿宋，行距28磅
3. **结束语**："谢谢大家！"单独成段
4. **落款**：单位、职务、姓名、日期依次排列

## 3. 讨论提纲

### 3.1 基本结构
```
[讨论主题]讨论提纲

一、讨论背景
   （一）[背景情况一]
   （二）[背景情况二]
       ...

二、讨论目标
   （一）[目标一]
   （二）[目标二]
       ...

三、讨论要点
   （一）[要点一]
        1. [子要点1]
        2. [子要点2]
   （二）[要点二]
        1. [子要点1]
        2. [子要点2]
       ...

四、预期成果
   （一）[成果一]
   （二）[成果二]
       ...

五、时间安排
   （一）[时间段一]：[讨论内容]
   （二）[时间段二]：[讨论内容]
       ...

六、参考材料
   （一）[材料一]
   （二）[材料二]
       ...
```

### 3.2 内容要求
1. **目标明确**：清楚说明讨论要达到的目标
2. **要点全面**：覆盖讨论的主要方面
3. **层次分明**：主次分明，逻辑清晰
4. **实用性强**：便于实际讨论操作

### 3.3 格式要求
1. **标题**：二号小标宋，居中
2. **各级标题**：依次使用一、(一)、1、(1)
3. **内容**：三号仿宋，行距28磅
4. **排版**：层次清晰，便于阅读

## 4. 工作汇报

### 4.1 基本结构
```
关于[工作内容]的工作汇报

[上级单位名称]：

一、工作进展情况
   （一）[工作方面一]
        1. [具体进展1]
            - 完成情况：[数据/百分比]
            - 主要成效：[具体描述]
        2. [具体进展2]
            ...
   （二）[工作方面二]
       ...

二、主要成绩和亮点
   （一）[成绩一]
        1. [具体表现]
        2. [影响/意义]
   （二）[成绩二]
       ...

三、存在的问题和困难
   （一）[问题一]
        1. [具体表现]
        2. [原因分析]
   （二）[问题二]
       ...

四、下一步工作计划
   （一）[计划一]
        1. [具体措施]
        2. [责任部门]
        3. [完成时限]
   （二）[计划二]
       ...

五、需要支持的事项
   （一）[支持一]：[具体内容]
   （二）[支持二]：[具体内容]
       ...

特此汇报。

[汇报单位名称]
[日期]
```

### 4.2 内容要求
1. **实事求是**：如实反映工作情况和成效
2. **数据支撑**：用具体数据说明工作进展
3. **问题坦诚**：不回避工作中存在的问题
4. **计划可行**：下一步工作安排切实可行

### 4.3 格式要求
1. **主送机关**：三号仿宋，顶格
2. **正文**：三号仿宋，行距28磅
3. **数据呈现**：重要数据用阿拉伯数字突出
4. **落款**：单位名称、日期右对齐

## 5. 通知公告

```

各部门/相关人员：

为[目的]，经研究，决定[事项]。现将有关事项通知如下：

一、[事项一]
   （一）[具体要求1]
   （二）[具体要求2]
       ...

二、[事项二]
   （一）[具体要求1]
   （二）[具体要求2]
       ...

三、工作要求
   （一）[要求一]
   （二）[要求二]
       ...

   （一）[其他需要说明的事项]

[单位名称]
[日期]
```

2. **要求具体**：工作要求可操作性强
3. **便于执行**：便于相关部门和人员落实
4. **语气正式**：体现组织权威性

### 5.3 格式要求
1. **标题**：二号小标宋，居中
2. **主送机关**：三号仿宋，顶格
3. **正文**：三号仿宋，行距28磅
4. **落款**：单位名称、日期右对齐

## 6. 请示报告

### 6.1 基本结构
```
关于[事项]的请示

[上级单位名称]：

一、请示事由
   （一）[背景情况]
   （二）[需要解决的问题]
   （三）[请示的具体事项]

二、事由分析
   （一）[必要性分析]
        1. [理由一]
        2. [理由二]
   （二）[可行性分析]
        1. [条件一]
        2. [条件二]
       ...

三、建议方案
   （一）[方案一]
        1. [具体内容]
        2. [预期效果]
        3. [所需资源]
   （二）[方案二]
       ...

四、请求事项
   （一）[请求一]：[具体内容]
   （二）[请求二]：[具体内容]
       ...

以上请示，请予审批。

[请示单位名称]
[日期]
```

### 6.2 内容要求
1. **事由充分**：请示事由理由充分
2. **分析透彻**：对问题的分析深入透彻
3. **方案可行**：提出的建议方案切实可行
4. **请求明确**：请求事项表述明确具体

### 6.3 格式要求
1. **标题**：二号小标宋，居中
2. **主送机关**：三号仿宋，顶格
3. **正文**：三号仿宋，行距28磅
4. **结束语**："以上请示，请予审批。"单独成段

## 通用写作规范

### 1. 语言规范
1. 使用规范的书面语
2. 避免口语化表达
3. 用词准确，表述严谨
4. 语气恰当，分寸适度

### 2. 格式规范
1. 遵循GB/T 9704-2012标准
2. 要素齐全，编排规范
3. 版面整洁，美观大方
4. 便于阅读和存档

### 3. 内容规范
1. 主题明确，重点突出
2. 事实准确，数据可靠
3. 论据充分，论证有力
4. 建议合理，措施可行

### 4. 结构规范
1. 层次分明，逻辑清晰
2. 过渡自然，衔接紧密
3. 详略得当，主次分明
4. 结构完整，首尾呼应
