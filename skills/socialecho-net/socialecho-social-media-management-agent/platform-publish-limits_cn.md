# 各平台发布内容与格式限制（参考）

本文说明在 SocialEcho 中通过 **发布 API / 发布流程** 对接各社媒平台时，系统侧校验的**文案、媒体数量、格式与尺寸**等规则摘要，便于研发与运营提前准备素材。

> **提示**
>
> - 下列限制基于当前产品内嵌规则整理；**各平台官方政策可能随时调整**，若与平台后台或发布失败提示不一致，**以平台方及系统实时提示为准**。
> - 文中「竖版」指**建议或强制竖屏素材**（通常理解为宽度 < 高度），具体见各小节。

---

## 术语说明


| 用语      | 含义                                                  |
| ------- | --------------------------------------------------- |
| 内容长度    | 正文字符数上限（含空格等，以系统校验为准）                               |
| 标题长度    | 标题字符数限制；「不限」表示未单独设硬性上限（仍可能受平台约束）                    |
| 媒体数量    | 单次发布可包含的图片 / GIF / 视频数量及合计规则                        |
| 混发规则    | 多种媒体类型同时发布时是否允许及组合限制                                |
| 纯文案     | 是否允许不带任何图片/视频/GIF 的发布                               |
| 需要分类 ID | 是否必须选择图版、社区等平台侧「分类/版块」类标识（如 Pinterest 图版、Reddit 社区） |
| 必须竖版    | 是否为竖屏素材要求（宽 < 高）                                    |

## Facebook

### Regular（常规动态 / `post` 等）


| 项目      | 限制                               |
| ------- | -------------------------------- |
| 内容长度    | 最大 2200 字符                       |
| 标题长度    | 不限                               |
| 最少标题/内容 | 不限                               |
| 媒体数量    | 图片最多 10；GIF 最多 1；视频最多 1；总个数不限    |
| 混发规则    | 图片 + GIF **不允许**；图片 + 视频 **不允许** |
| 纯文案     | **允许**                           |
| 图片格式    | jpg / jpeg / png / gif           |
| 视频格式    | mp4 / mov                        |
| 图片大小    | 静态图 8MB；GIF 8MB                  |
| 视频大小    | 最大 1GB                           |
| 视频时长    | 1 秒～20 分钟                        |
| 视频分辨率   | 不限                               |
| 图片分辨率   | 不限                               |
| 帧率      | 23～60 fps                        |
| 视频/图片比例 | 不限                               |
| 评论长度    | 不限                               |
| 需要分类 ID | 否                                |
| 必须竖版    | 否                                |
