## ✨ 功能亮点

| 功能 | 描述 |
|------|------|
| 🧠 **智能分析** | 综合考虑每位成员的偏好，自动计算匹配度 |
| 🗳️ **民主决策** | 支持"一票否决"机制，照顾特殊需求成员 |
| 🔍 **实时数据** | 通过 fly.ai 获取景点、酒店、门票实时信息 |
| 📊 **可视化输出** | 自动生成精美 HTML 报告，方便分享 |
| 👨‍👩‍👧‍👦 **亲子友好** | 支持带孩子场景，推荐适合全家的目的地 |

## 🚀 快速开始

### 安装

**方式一：从 ClawHub 安装**
```bash
clawhub install team-outing-planner
```

**方式二：手动安装**
```bash
# 复制到 Qoder skills 目录
cp -r team-outing-planner ~/.qoder/skills/
```

### 使用

直接在 Qoder 中输入团建需求：

```
帮我规划公司团建，团队20人，从上海出发，预算800元/人
```

或使用 slash 命令：

```
/team-outing-planner 15人从杭州出发，想去海边，预算1200元
```

## 🎯 支持的偏好类型

### 活动类型

| 类型 | 图标 | 适合场景 |
|------|------|----------|
| 自然风光 | 🏞️ | 喜欢户外、亲近自然 |
| 历史古迹 | 🏛️ | 文化爱好者、研学团队 |
| 主题乐园 | 🎢 | 年轻团队、亲子活动 |
| 温泉度假 | ♨️ | 放松休闲、冬季团建 |
| 户外运动 | ⛰️ | 体力好、喜欢挑战 |
| 城市观光 | 🌃 | 轻松游览、美食购物 |

### 运动强度

| 强度 | 说明 | 示例目的地 |
|------|------|------------|
| 轻松休闲 | 步行少，观光为主 | 温泉酒店、海滨度假 |
| 适度活动 | 有一定步行量 | 古镇游览、主题乐园 |
| 挑战型 | 户外运动为主 | 徒步、漂流、滑雪 |

### 特殊需求处理

| 需求 | 处理方式 |
|------|----------|
| 🦽 行动不便 | 优先无障碍设施完善的目的地 |
| 😰 恐高 | 排除高空项目（玻璃栈道、蹦极等） |
| 🤢 晕车 | 优先近距离或高铁直达 |
| 🥗 饮食限制 | 推荐餐饮选择丰富的目的地 |
