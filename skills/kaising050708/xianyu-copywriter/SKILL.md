# 电商文案生成器

批量生成电商产品文案，支持淘宝/拼多多/闲鱼/京东，包含标题 + 详情 + 卖点。

## When to Activate

Trigger when the user mentions: 电商文案，产品描述，淘宝标题，闲鱼文案，商品详情.

## Workflow

1. **Confirm inputs:**
   - 产品名称/类型 (required)
   - 目标平台 (taobao/xianyu/pinduoduo/jd)
   - 价格区间 (optional)
   - 目标人群 (optional: 学生党/上班族/宝妈/等)
   - 文案数量 (default: 5, max: 50)
   - 风格偏好 (optional: 专业型/种草型/促销型/故事型)

2. **Generate content by platform:**

### 淘宝/天猫标题 (30 字内)
```
{核心关键词} {属性词} {卖点词} {场景词}
示例：2026 新款夏季透气运动鞋男跑步鞋轻便减震健身鞋
```

### 拼多多标题 (强调价格/促销)
```
{促销词}+{核心词}+{规格}+{卖点}
示例：【买一送一】夏季冰丝 T 恤男短袖透气速干运动上衣
```

### 闲鱼文案 (个人卖家风格)
```
{成色描述}
{入手渠道}
{转手原因}
{产品亮点}
{价格优势}
{交易方式}
```

### 产品详情页 (5 段式)
```
【痛点引入】- 描述用户痛点
【产品介绍】- 核心功能和参数
【卖点展示】- 3-5 个核心卖点
【使用场景】- 具体使用场景
【信任背书】- 售后保障/用户评价
```

3. **SEO 优化:**
   - 包含搜索热词
   - 避免违禁词 (最/第一/唯一等)
   - 关键词自然分布

4. **转化优化:**
   - 紧迫感 (限时/限量)
   - 从众心理 (已售/好评)
   - 风险逆转 (包邮/退换)
