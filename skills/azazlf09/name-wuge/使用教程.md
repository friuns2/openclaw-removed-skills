## 1. 测名（最常用）

发送姓名即可，不需要复杂指令：

| 使用场景 | 直接发送 |
|---------|---------|
| 测单字名 | `李明` |
| 测双字名 | `陈星任` |
| 带指令 | `测名 张伟` / `算命 李云帆` / `五格 王芳` |
| 提取姓名 | `帮我看看这个名字好不好：陈浩然` |

**输出示例：**
```
-------------------------------------
  name: 陈星任（陳星任）      source: full kangxi
  strokes: 陳=16画  星=9画  任=6画  (康熙繁体笔画)
-------------------------------------
  格      数理    吉凶     解释
  天格    17    吉       排除万难，有贵人助
  人格    25    吉       天时地利，名利双收
  地格    15    吉       谦恭做事，外得人和
  外格    7     吉       独营生意，和气吉祥
  总格    31    吉       能得人望，排除万难
-------------------------------------
  overall score:  89.0 / 100
-------------------------------------
```

## 3. 批量生成名字

| 想要什么 | 直接发送 |
|---------|---------|
| 姓氏取名 | `批量取名 石` / `生成名字 李` / `帮我想名字` |
| 五格全吉版 | `生成名字 王 五格全吉` |
| 指定名字长度 | `生成名字 李 单名` / `生成名字 张 两字名` |
| 公司名生成 | `生成公司名 3字 总格25` |
| 纯随机好听名字 | `随机生成10个好听的名字` |

**默认规则：总格为吉（其他四格随机）**
**五格全吉：五格全部为吉**

## 注意事项

- **繁简转换**：简体/繁体混写都能识别，自动转繁体查康熙笔画
- **复姓支持**：欧阳、司马等自动识别
- **生僻字**：自动下载完整康熙字典（约3MB），无需手动操作
- **评分标准**：总格为吉是核心，其他四格不强制要求
