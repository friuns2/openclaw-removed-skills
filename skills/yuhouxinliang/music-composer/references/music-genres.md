## ⚡ 节奏与速度（BPM Guide）

| 速度术语 | BPM | 情绪描述 |
|---------|-----|---------|
| Grave（庄板）| 25-45 | 庄严、沉重 |
| Largo（广板）| 40-60 | 宽广、肃穆 |
| Adagio（慢板）| 66-76 | 缓慢、抒情 |
| Andante（行板）| 76-108 | 步调适中 |
| Moderato（中板）| 108-120 | 温和、从容 |
| Allegro（快板）| 120-156 | 轻快、活泼 |
| Vivace（活板）| 140-176 | 快速、兴奋 |
| Presto（急板）| 168-200 | 极快、激烈 |

**场景推荐 BPM：**
- 睡眠/冥想：40-60
- 咖啡馆背景：70-90
- 工作/学习：80-100
- 走路/跑步：110-140
- 派对/运动：130-160

## 🎹 乐器编配（Instrumentation）

### 弦乐组
- Violin（小提琴）— 明亮、高亢
- Viola（中提琴）— 温暖、中音
- Cello（大提琴）— 浑厚、低沉
- Double Bass（低音提琴）— 厚实、爵士基础
- Harp（竖琴）— 梦幻、点缀

### 键盘组
- Grand Piano（三角钢琴）— 古典/爵士/流行核心
- Electric Piano（电钢琴）— 爵士/灵魂乐常用
- Organ（风琴）— 教堂/布鲁斯/摇滚
- Synthesizer（合成器）— 电子/氛围/赛博朋克

### 吉他组
- Acoustic Guitar（原声吉他）— 民谣/流行基础
- Electric Guitar（电吉他）— 摇滚/电子
- Fingerpicking Guitar（指弹吉他）— 古典/民谣
- Jazz Guitar（爵士吉他）— 和弦丰富

### 打击乐组
- Drums/Trap Drums（鼓组）— 摇滚/电子/流行
- Cajon（卡洪鼓）— 弗拉明戈/民谣
- Djembe（非洲鼓）— 部落/世界音乐

### 民族乐器
- Guzheng（古筝）— 中国古典
- Erhu（二胡）— 忧伤、苍凉
- Chinese Flute/Bamboo Flute（笛子）— 中国风

## ✨ 活人感·过渡·层次·情感——关键词词库

### 🫁 活人感（Human Feel）必须词
| 中文含义 | 英文 Prompt 关键词 |
|---------|-----------------|
| 真实乐器感 | `organic instruments`, `real instruments` |
| 轻微呼吸感 | `breathable`, `breath space` |
| 轻微不完美 | `subtle imperfection`, `human touch` |
| 轻微摇摆 | `slight swing`, `groove feel`, `laid-back` |
| 自然动态 | `natural dynamics`, `dynamic variation` |
| 录音棚空间感 | `room ambience`, `recording studio` |
| 模拟真实演奏 | `live performance feel`, `realistic performance` |
| 手指/弓弦噪音 | `piano key noise`, `bow friction`（钢琴/弦乐时）|
| 呼吸声/人声气声 | `breathy vocal`, `intimate whisper` |

### 🌊 乐器过渡（Instrument Transitions）
| 中文含义 | 英文 Prompt 关键词 |
|---------|-----------------|
| 平滑过渡 | `smooth instrument transitions`, `seamless layering` |
| 渐进引入 | `fade in gently`, `slowly introduced` |
| 渐隐淡出 | `gradual fade out`, `tail off naturally` |
| 乐器互换 | `instruments fade into each other` |
| 层叠推进 | `layered build-up` |
| 情绪引导 | `emotionally-driven transitions` |
| 呼吸间隔 | `pauses between phrases`, `breathing room` |

### 🧅 层次感（Layering）必须词
| 中文含义 | 英文 Prompt 关键词 |
|---------|-----------------|
| 分层编曲 | `layered arrangement`, `multi-layered` |
| 前景清晰 | `foreground: melodic lead`, `clear lead line` |
| 中景支撑 | `midground: harmonic support` |
| 背景铺垫 | `background: ambient pad`, `textural layer` |
| 频率分离 | `well-separated frequency layers` |
| 立体声场 | `wide stereo field`, `spatial depth` |
| 动态对比 | `dynamic contrast between sections` |
| 留白呼吸 | `space and silence`, `rests for breathe` |

### 💗 情感细腻（Emotional Depth）
| 中文含义 | 英文 Prompt 关键词 |
|---------|-----------------|
| 细腻情感 | `tender emotional depth`, `nuanced emotion` |
| 情感起伏 | `emotional arc`, `swell and release` |
| 温暖真诚 | `warm and sincere`, `intimate` |
| 忧而不伤 | `melancholic but beautiful` |
| 克制表达 | `understated`, `restrained passion` |
| 共鸣感 | `resonant`, `soul-stirring` |
| 泪水感 | `emotionally overwhelming`, `goosebump` |
| 真诚感 | `raw emotion`, `genuine` |

### 🚫 避免的词（会让音乐太"假"）
| 避免 | 原因 |
|------|------|
| `perfect`, `flawless` | 太完美=假 |
| `electronic cold` | 冰冷无温度 |
| `flat dynamics` | 无起伏=无聊 |
| `stiff rhythm` | 节拍僵硬 |
| `overproduced` | 过度制作=塑料感 |
| `robotic`, `synthetic` | 机器人感 |
