# Lyrics Structure Templates

## 通用歌词格式说明

```
[verse]         — 主歌：铺垫故事/情感
[pre-chorus]    — 预副歌：情绪上升
[chorus]        — 副歌：高潮/核心hook
[bridge]        — 桥段：情绪转折/对比
[outro]         — 结尾：收束
[intro]         — 前奏（可选）
```
