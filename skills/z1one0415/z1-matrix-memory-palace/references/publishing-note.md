# Publishing Note

## Required source attribution
This skill is inspired by and explicitly references:
- Andrej Karpathy's llm-wiki concept: persistent LLM-maintained wiki as a compounding middle layer between raw sources and answers.
- Jeff Pierce's memory-palace approach: long-term model-external memory layer with structured routing and multi-instance potential.

## Fusion statement
This skill combines:
- Memory Palace as the spatial shell and routing bus
- LLM Wiki as the continuous synthesis and reflection soul
- File-driven command discipline as the operational constitution

## Recommended user-facing summary
“Z1矩阵宫殿记忆法”融合了 Karpathy 的 llm-wiki 思想与 Jeff Pierce 的 memory-palace 思想：前者解决知识如何持续编译，后者解决记忆如何独立于模型长期存在；再叠加文件驱动、单主链、低 token 的调度宪法，使多 Agent 协作从聊天堆栈转向物理可验收的记忆总线。
