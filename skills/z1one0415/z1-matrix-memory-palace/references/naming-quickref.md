# Naming Quick Reference

## Corridor cards
- file: `YYYY-MM-DD_<slug>.md`
- task_id: `<scope>-<slug>-<date>`

## Project rooms
- dir: `<project_slug>/`
- files: README / index / reflection_backfill / chain_map(optional)

## Reflection pages
- file: `<semantic_slug>_<YYYY-MM-DD>.md`

## Core protocols
- file: `z1_<protocol_name>_v<major>_<YYYY-MM-DD>.md`

## Agent runbooks
- file: `RUNBOOK_V<major>_<YYYY-MM-DD>.md`
