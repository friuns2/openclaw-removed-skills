# 产品研发全流程操作指引

## 任务目标
- 本技能用于：指导产品研发全流程，提供阶段递进提醒和操作指引
- 核心能力：阶段导航、上步总结、下步提醒、注意事项
- 触发条件：研发项目启动、阶段转换、流程不清晰时

## 研发全流程阶段

```
市场调研 → 概念设计 → 配方研发 → 工艺开发 → 小试 → 中试 → 
感官评价 → 稳定性测试 → 包材设计 → 法规审核 → 产品定型 → 
送检认证 → 量产准备 → 上市推广
```

## 各阶段指引

### 1. 市场调研阶段

#### 本阶段目标
- 确定目标市场和消费者需求
- 分析竞品现状和市场机会
- 明确产品定位和差异化方向

#### 上一步提醒
**（项目启动，无上一步）**

#### 本阶段操作要点
1. 明确目标人群和使用场景
2. 分析市场规模和增长趋势
3. 调研竞品价格、配方、卖点
4. 识别市场空白和机会点

#### 注意事项
- 避免闭门造车，确保市场需求真实存在
- 关注消费者真实反馈，而非主观假设
- 竞品分析要全面，包括线上线下渠道

#### 下一步提醒
进入**概念设计**前需确认：
- [ ] 目标人群画像明确
- [ ] 市场规模可支撑项目立项
- [ ] 差异化方向清晰
- [ ] 初步价格带定位

#### 常用工具
```
web_search(query="XX品类 市场规模 消费者画像")
web_search(query="XX竞品 配方 卖点 分析")
web_search(query="XX品类 趋势 机会点 新品")
```

### 2. 概念设计阶段

#### 本阶段目标
- 确定产品核心概念和卖点
- 制定产品差异化策略
- 明确开发目标和技术路线

#### 上一步总结
**市场调研阶段需输出：**
- 目标人群定义
- 市场机会分析报告
- 竞品对比表
- 产品定位建议

#### 本阶段操作要点
1. 提炼产品核心概念
2. 确定主诉求和差异化卖点
3. 设定产品关键指标目标
4. 制定初步配方方向

#### 注意事项
- 概念要具体可感知，避免空洞口号
- 卖点需有依据支撑
- 设定指标要兼顾可行性和竞争力

#### 下一步提醒
进入**配方研发**前需确认：
- [ ] 产品概念清晰（一句话说明）
- [ ] 主卖点明确（不超过3个）
- [ ] 目标成本范围
- [ ] 关键指标目标值

#### 常用工具
```
web_search(query="无糖饼干 概念设计 卖点")
web_search(query="代餐 产品概念 差异化 策略")
```

### 3. 配方研发阶段

#### 本阶段目标
- 开发满足概念的配方
- 优化配方成本和稳定性
- 确定各原料规格要求

#### 上一步总结
**概念设计阶段需输出：**
- 产品概念书
- 卖点梳理文档
- 目标成本范围
- 关键指标目标

#### 本阶段操作要点
1. 筛选原料和添加剂
2. 进行基础配方设计
3. 感官初步调试
4. 小批量试制验证

#### 注意事项
- 原料选择要考虑供应稳定性和成本
- 添加剂使用必须符合GB 2760
- 配方调整要记录每次变更原因
- 小样测试要有代表性

#### 下一步提醒
进入**工艺开发**前需确认：
- [ ] 基础配方稳定可重复
- [ ] 感官评价达标
- [ ] 成本在目标范围内
- [ ] 原料规格书已确定

#### 常用工具
```
web_search(query="XX原料 规格 要求 食品级")
web_search(query="XX添加剂 用量 限量 GB 2760")
web_search(query="XX品类 配方 优化 技巧")
```

---

### 4. 工艺开发阶段

#### 本阶段目标
- 确定最佳工艺参数
- 建立工艺标准操作流程
- 验证工艺稳定性和可重复性

#### 上一步总结
**配方研发阶段需输出：**
- 锁定配方（BOM清单）
- 原料规格书
- 感官评价报告
- 初步成本核算

#### 本阶段操作要点
1. 梳理完整工艺流程
2. 识别关键控制点（CCP）
3. 优化工艺参数
4. 编写SOP文件

#### 注意事项
- 工艺参数要有科学依据
- 关键参数要经过验证
- 工艺变更需评估影响
- 做好工艺记录追溯

#### 下一步提醒
进入**小试**前需确认：
- [ ] 工艺流程图已绘制
- [ ] 关键参数已确定
- [ ] SOP初稿已完成
- [ ] 工艺风险评估通过

#### 常用工具
```
web_search(query="XX产品 工艺流程 关键控制点")
web_search(query="食品工艺 SOP 编写规范")
web_search(query="XX设备 工艺参数 参考")
```

---

### 5. 小试阶段

#### 本阶段目标
- 在实验室规模验证配方和工艺
- 获得小批量样品
- 为中试提供参数依据

#### 上一步总结
**工艺开发阶段需输出：**
- 工艺流程图
- 关键参数清单
- SOP文件
- 风险评估报告

#### 本阶段操作要点
1. 按照SOP进行小批量试制
2. 收集工艺参数数据
3. 样品感官评估
4. 记录问题并优化

#### 注意事项
- 小试要模拟真实生产条件
- 参数记录要完整详细
- 问题要及时记录和追溯
- 样品要有足够数量供测试

#### 下一步提醒
进入**中试**前需确认：
- [ ] 小试工艺稳定可重复
- [ ] 样品质量符合预期
- [ ] 工艺参数已优化
- [ ] 问题清单已关闭

---

### 6. 中试阶段

#### 本阶段目标
- 在接近生产规模验证可行性
- 确认生产设备适配性
- 获取量产参数依据

#### 上一步总结
**小试阶段需输出：**
- 小试报告
- 工艺参数优化方案
- 样品检测报告
- 问题整改记录

#### 本阶段操作要点
1. 联系中试车间或代工厂
2. 进行放大试制
3. 收集生产数据
4. 评估量产可行性

#### 注意事项
- 中试规模要有代表性
- 设备差异要提前沟通
- 批量间差异要监控
- 成本数据要真实记录

#### 下一步提醒
进入**感官评价**前需确认：
- [ ] 中试批量不少于3批
- [ ] 批次间差异可控
- [ ] 成本数据已核算
- [ ] 产能评估已完成

---

### 7. 感官评价阶段

#### 本阶段目标
- 组织专业感官评价
- 收集消费者测试反馈
- 为配方优化提供依据

#### 上一步总结
**中试阶段需输出：**
- 中试报告
- 批量生产数据
- 成本核算表
- 产能评估报告

#### 本阶段操作要点
1. 设计感官评价方案
2. 组建评价小组或消费者测试
3. 收集多维度感官数据
4. 分析结果提出改进建议

#### 注意事项
- 评价环境要标准化
- 样品制备要规范一致
- 评价员要经过培训
- 数据收集要客观真实

#### 下一步提醒
进入**稳定性测试**前需确认：
- [ ] 感官评价达标
- [ ] 需改进项已优化
- [ ] 最终配方已锁定
- [ ] 评价报告已归档

#### 常用工具
```
web_search(query="食品感官评价 标准 方法")
web_search(query="消费者测试 问卷设计")
web_search(query="XX产品 感官指标 评价标准")
```

---

### 8. 稳定性测试阶段

#### 本阶段目标
- 验证产品保质期
- 确定存储条件
- 确保产品安全

#### 上一步总结
**感官评价阶段需输出：**
- 感官评价报告
- 消费者测试报告
- 配方调整记录
- 最终配方确认书

#### 本阶段操作要点
1. 制定稳定性测试方案
2. 按计划进行存储测试
3. 定期检测各项指标
4. 分析数据确定保质期

#### 注意事项
- 测试条件要符合标准
- 检测方法要一致
- 记录要完整可追溯
- 异常情况要及时分析

#### 下一步提醒
进入**包材设计**前需确认：
- [ ] 加速试验完成
- [ ] 保质期已确定
- [ ] 存储条件已明确
- [ ] 检测报告已归档

#### 常用工具
```
web_search(query="食品保质期试验 标准 方法")
web_search(query="Q10法 保质期计算")
web_search(query="XX产品 稳定性 指标 要求")
```

### 9. 包材设计阶段

#### 本阶段目标
- 设计产品包装方案
- 选型包材规格
- 完成标签文案

#### 上一步总结
**稳定性测试阶段需输出：**
- 稳定性测试报告
- 保质期声明
- 存储条件说明
- 各项指标检测结果

#### 本阶段操作要点
1. 确定包装形式和规格
2. 选择包材类型和供应商
3. 设计包装展开图
4. 编写标签文案

#### 注意事项
- 包材要满足保护功能
- 食品接触材料要合规
- 标签要符合GB 7718
- 设计要考虑生产和物流

#### 下一步提醒
进入**法规审核**前需确认：
- [ ] 包装规格已确定
- [ ] 包材已选型打样
- [ ] 标签文案已初稿
- [ ] 包装成本已核算

#### 常用工具
```
web_search(query="食品包装材料 标准 GB 4806")
web_search(query="预包装食品标签 规范 GB 7718")
web_search(query="XX产品 包装 设计 范例")
```

### 10. 法规审核阶段

#### 本阶段目标
- 确保产品符合法规要求
- 完成必要的注册备案
- 规避合规风险

#### 上一步总结
**包材设计阶段需输出：**
- 包装设计稿
- 包材规格书
- 标签文案
- 包装成本表

#### 本阶段操作要点
1. 核查配方和添加剂合规性
2. 审核标签标识合规性
3. 确认产品标准适用性
4. 办理必要注册备案

#### 注意事项
- 关注最新法规动态
- 特殊产品有额外要求
- 标签审核要仔细
- 必要时寻求专业意见

#### 下一步提醒
进入**产品定型**前需确认：
- [ ] 配方合规审查通过
- [ ] 标签审核通过
- [ ] 标准适用确认
- [ ] 注册备案完成（如需）

#### 常用工具
```
web_search(query="食品生产许可 审查细则")
web_search(query="XX产品 执行标准 GB")
web_search(query="特殊食品 注册 备案 要求")
```

### 11. 产品定型阶段

#### 本阶段目标
- 锁定产品全部技术文件
- 确认可量产状态
- 完成内部审批流程

#### 上一步总结
**法规审核阶段需输出：**
- 合规审核报告
- 标准适用确认函
- 注册备案证明（如需）

#### 本阶段操作要点
1. 汇总全部技术文件
2. 组织定型评审会
3. 确认量产条件
4. 完成内部审批

#### 注意事项
- 文件要完整准确
- 评审要全面严格
- 变更通道要关闭
- 责任要明确到人

#### 下一步提醒
进入**送检认证**前需确认：
- [ ] 技术文件包完整
- [ ] 定型评审通过
- [ ] 量产条件具备
- [ ] 内部审批完成

---

### 12. 送检认证阶段

#### 本阶段目标
- 获得第三方检测报告
- 验证产品符合标准
- 为上市销售做准备

#### 上一步总结
**产品定型阶段需输出：**
- 完整技术文件包
- 定型评审报告
- 量产确认书

#### 本阶段操作要点
1. 选择检测机构
2. 准备送检样品和资料
3. 跟进检测进度
4. 获取检测报告

#### 注意事项
- 选择资质齐全的机构
- 样品要有代表性
- 检测项目要全面
- 报告要及时获取

#### 下一步提醒
进入**量产准备**前需确认：
- [ ] 检测报告已获取
- [ ] 检测结果全部合格
- [ ] 不合格项已整改
- [ ] 报告已归档

#### 常用工具
```
web_search(query="食品第三方检测机构 推荐")
web_search(query="XX产品 检测项目 标准")
web_search(query="检测报告 周期 费用")
```

### 13. 量产准备阶段

#### 本阶段目标
- 完成量产导入
- 确认供应链就绪
- 建立质量控制体系

#### 上一步总结
**送检认证阶段需输出：**
- 第三方检测报告
- 合格证明文件

#### 本阶段操作要点
1. 完成生产导入会议
2. 确认原料供应就绪
3. 验证生产设备
4. 建立QC标准

#### 注意事项
- 量产前要有试产
- 供应链要提前布局
- 质量标准要明确
- 人员培训要到位

#### 下一步提醒
进入**上市推广**前需确认：
- [ ] 量产导入完成
- [ ] 首批成品合格
- [ ] 供应链稳定
- [ ] 质量体系运行

---

### 14. 上市推广阶段

#### 本阶段目标
- 产品成功上市
- 建立市场认知
- 实现销售目标

#### 上一步总结
**量产准备阶段需输出：**
- 量产确认报告
- 首批成品报告
- 供应链清单
- QC标准文件

#### 本阶段操作要点
1. 制定上市计划
2. 准备推广物料
3. 铺货和销售
4. 收集市场反馈

#### 注意事项
- 上市计划要周全
- 库存要合理备货
- 渠道要充分沟通
- 反馈要及时响应

## 阶段检查清单模板

### 阶段完成标准
每个阶段完成前，请确认以下内容：

1. **产出物完整性**
   - [ ] 阶段产出物清单已列出
   - [ ] 每项产出物已审核确认
   - [ ] 产出物已归档保存

2. **决策点通过**
   - [ ] 下一步骤进入条件已明确
   - [ ] 决策评审已通过
   - [ ] 问题已关闭或记录

3. **知识传递**
   - [ ] 关键信息已传递给下阶段
   - [ ] 风险已识别和沟通
   - [ ] 建议已记录
