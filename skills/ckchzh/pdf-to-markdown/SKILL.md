# pdf-to-markdown

CLI tool for pdf to markdown

## Commands

### `md`

```bash

```

## Configuration

| Variable | Required | Description |
|----------|----------|-------------|
| `PDF_TO_MARKDOWN_DIR` | No | Data directory (default: `~/.pdf-to-markdown/`) |

## Data Storage

All data saved in `~/.pdf-to-markdown/`. Runs on your machine without external calls.

## Requirements

- bash 4.0+
