# cli-obsidian

**Obsidian 命令行工具 - 让 AI Agent 直接操作你的笔记**

## 功能

- 命令行创建/搜索/管理笔记
- 支持标签系统
- JSON 输出，完美对接 AI Agent
- 跨平台支持（Win/Mac/Linux）

## 安装

```bash
pip install cli-obsidian
```

## 使用

```bash
# 创建笔记
cli-obsidian create "会议记录" --tag work

# 搜索笔记
cli-obsidian search "AI Agent"

# 列出笔记
cli-obsidian list --tag work
```
