# CLI-Obsidian - Obsidian 命令行工具

让 AI Agent 直接操作你的 Obsidian 笔记！

## 核心功能

- ✅ 命令行创建/搜索/管理笔记
- ✅ 支持标签系统
- ✅ JSON 输出，完美对接 AI
- ✅ 跨平台支持

## 使用场景

- AI Agent 自动记录会议笔记
- 脚本批量管理笔记
- 快速搜索查找
- 数据导出

## 安装

```bash
pip install cli-obsidian
```

## 快速开始

```bash
# 创建笔记
cli-obsidian create "会议记录" --tag work

# 搜索笔记
cli-obsidian search "AI Agent"

# 列出笔记
cli-obsidian list --tag work
```
