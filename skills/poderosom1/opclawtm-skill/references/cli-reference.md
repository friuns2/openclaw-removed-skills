# CLI 命令参考

> 你是 AI Agent，此文档列出你可执行的查询命令和 TUI 操作说明。

---

## 命令分类

### 可执行的命令（你可以直接执行）

这些查询类命令你可以直接帮助用户执行：

| 命令 | 用途 | 执行时机 |
|------|------|----------|
| `opclawtm --version` | 查看版本 | 验证安装 |
| `opclawtm license-status` | 查看授权状态 | 用户询问授权信息 |
| `opclawtm list-agents` | 列出所有 Agent | 查询 Agent 信息 |
| `opclawtm list-agents --role assistant` | 查询总助理 ID | 私有 Skill 创建前 |
| `opclawtm trial-activate` | 试用激活 | 新用户首次使用 |
| `opclawtm bind-license --key XXX` | 绑定授权 | 用户有测试码 |

### 必须由用户在 TUI 中完成的操作

这些操作**你不能代替用户执行**，必须引导用户打开 TUI 自己操作：

| 操作 | TUI 路径 |
|------|----------|
| 创建团队 | 团队管理 → 创建团队 |
| 创建 Agent | Agent 管理 → 创建 Agent |
| 飞书 Bot 创建 | 飞书配置 → 为 Agent 添加飞书 Bot |
| 群绑定 | 飞书配置 → 绑定飞书群到事业部 |
| 用户绑定 | 飞书配置 → 用户绑定 |
| 任务创建 | 任务管理 → 创建任务 |

---

## TUI 操作说明

### 引导用户启动 TUI

**告诉用户：** "现在需要您打开 TUI 界面进行操作，请执行以下命令："

```bash
opclawtm tui
```

### TUI 功能入口

| 功能 | TUI 路径 |
|------|----------|
| 授权绑定 | 主菜单 → 授权管理 |
| 团队创建 | 主菜单 → 团队创建向导 |
| 团队查看 | 主菜单 → 团队查看 |
| 飞书配置 | 主菜单 → 飞书配置 |
| Bot 绑定 | 飞书配置 → Bot 绑定 |
| 群绑定 | 飞书配置 → 🔗 绑定飞书群到事业部 |
| 用户绑定 | 飞书配置 → 用户绑定 |
| 配对管理 | 飞书配置 → 配对管理 |

---

## 操作原则

### ID 输入
- **所有 ID 直接回车自动填充**
- 不要手动输入 ID
- 自动生成的 ID 是英文格式

### 团队创建
- 使用「团队创建向导」
- 不使用单独的创建命令

### 飞书配置
- 在 TUI 中完成配置
- 不使用命令行配置

---

## 错误处理

| 错误信息 | 原因 | 你的行动 |
|----------|------|----------|
| "系统未初始化" | 首次启动 | 引导用户完成初始化 |
| "授权无效" | 未绑定授权 | 引导用户激活授权 |
| "ID 已存在" | 重复创建 | 引导用户使用不同名称 |

## 数据存储位置

| 系统 | 路径 |
|------|------|
| Windows | `C:\Users\{用户名}\.openclaw\` |
| macOS/Linux | `~/.openclaw/` |

| 目录 | 说明 |
|------|------|
| `logs/` | 日志文件（查找 ou_ ID）|
| `workspace/skills/` | Skill 存放位置 |
