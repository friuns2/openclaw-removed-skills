## 📊 十维行业分析框架（全维度深度分析）

### 默认分析深度：深度分析 ⭐️⭐️⭐️

| 分析深度 | 输出长度 | 适用场景 |
|----------|----------|----------|
| **深度分析** | 10000-20000 字 | 首次覆盖、重点行业、投资决策 |
| 快速分析 | 3000-5000 字 | 日常跟踪、简单查询 |

### 十维全部深度展开（无简化章节）

| 维度 | 权重 | 优先级 | 核心内容 | 数据源 |
|------|------|--------|----------|--------|
| 1️⃣ **行业概览** | 10% | P0 | 定义、分类、市场规模、增速 | mx-search + openclaw-tavily-search |
| 2️⃣ **业务内容** | 12% | P0 | 主要产品/服务、商业模式、盈利模式 | mx-search |
| 3️⃣ **发展阶段** | 10% | P0 | 导入期/成长期/成熟期/衰退期判断 | mx-search + openclaw-tavily-search |
| 4️⃣ **市场前景** | 12% | P0 | 市场空间、增速预测、驱动因素、细分市场 | mx-search + openclaw-tavily-search |
| 5️⃣ **产业链拆解** | 15% | P0 | 上游/中游/下游、核心环节、价值分布、壁垒分析 | mx-search + openclaw-tavily-search |
| 6️⃣ **成本结构** | 12% | P0 | 核心部件成本占比、毛利率分析、成本趋势 | mx-search + 年报 |
| 7️⃣ **竞争格局** | 10% | P0 | CR5/CR10、市场份额、竞争态势 | mx-search |
| 8️⃣ **核心公司** | 8% | P0 | 龙头公司深度分析、技术优势、市占率、财务对比 | mx-data + mx-search |
| 9️⃣ **政策环境** | 6% | P0 | 支持/限制政策、监管趋势 | openclaw-tavily-search |
| 🔟 **风险与机遇** | 5% | P0 | 主要风险、投资机会 | mx-search + openclaw-tavily-search |
| **综合** | **100%** | - | - | - |

## 🛠️ 工具调用规范

### 必用工具（P0 优先级）

| 工具 | 用途 | 时效性要求 | 调用说明 |
|------|------|------------|----------|
| **mx-search** | 最新新闻、公告、研报 | **15 天内优先，核心数据 7 天内** | 获取行业最新动态、机构观点 |
| **openclaw-tavily-search** | 全网深度搜索、行业数据、政策文件 | 30 天内 | 补充行业数据、政策、竞争格局 |

### 辅助工具（P1 优先级）

| 工具 | 用途 | 时效性要求 |
|------|------|------------|
| **mx-data** | 行业内上市公司财务数据、行情 | 实时 |
| **mx-xuangu** | 行业成分股查询、股票池筛选、板块资金流向 | 实时 |
| **妙想资讯** | 行业数据、产业链信息 | 30 天内 |

### 数据交叉验证原则

```
重要数据必须至少 2 个独立来源验证：
├── 市场规模 → 多份研报交叉 + 行业协会数据
├── 成本占比 → 招股书 + 年报 + 调研纪要
├── 市占率 → 多份研报 + 行业协会 + 公司官网
├── 政策文件 → 政府官网 + 官方媒体
├── 技术趋势 → 专利数据 + 专家访谈 + 研报
└── 成分股数据 → mx-xuangu + 多份研报交叉
```

## ⚠️ 注意事项

### 必须执行（数据质量优先）

1. ✅ **所有数据标注来源和日期** - 每个数据都要有来源
2. ✅ **重要数据交叉验证（至少 2-3 源）** - 市场规模/增速等核心数据
3. ✅ **表格增加"数据来源"列** - 所有表格都要有
4. ✅ **研报 15 天内优先，核心数据 7 天内** - 灵活处理时效性
5. ✅ **生成飞书文档** - 必须生成
6. ✅ **使用 mx-xuangu 查询成分股** - 确保公司分析准确性
7. ✅ **核心逻辑基于最新数据** - 不用过时数据支撑观点
8. ✅ **内容深度优先** - 不追求形式，追求内容质量

**研报时效性灵活处理：**
- **核心数据**（市场规模、增速、竞争格局）：优先 7 天内，可放宽至 15 天
- **行业趋势、政策分析**：可放宽至 30 天
- **历史对比、长期逻辑**：可使用更早期研报（需标注日期）

### 禁止行为

1. ❌ **使用过时数据（核心数据研报>30 天）** - 除非是历史对比
2. ❌ **单一来源核心数据** - 必须交叉验证
3. ❌ **无来源标注** - 所有数据都要标注
4. ❌ **简化分析章节** - 十维全部展开
5. ❌ **不生成飞书文档** - 必须生成存档
6. ❌ **形式化评分卡** - 移除综合评分卡，注重实质分析
7. ❌ **空洞的观点** - 所有观点都要有数据支撑

### 内容质量要求

| 维度 | 要求 | 执行方式 |
|------|------|---------|
| **数据最新** | **研报 15 天内优先，财报最新季报** | mx-search 时间灵活过滤 |
| **数据准确** | 官方公告 + 权威研报交叉验证 | 多源对比，差异>10% 需说明 |
| **逻辑严密** | 所有观点有数据支撑 | 避免"可能/也许/大概" |
| **分析深度** | 不止于表面，深入本质 | 多问"为什么"，挖掘根本原因 |
| **实用性** | 投资建议可执行 | 明确目标价、止损位、仓位建议 |
