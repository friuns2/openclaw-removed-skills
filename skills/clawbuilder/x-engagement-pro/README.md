# X Engagement Pro

OpenClaw skill for automated X (Twitter) engagement with AI image generation communities.

## Source

https://github.com/notphilsledge/x-engagement-pro

## Security

This skill requires:
- `X_API_KEY` — X API credentials (environment variable)
- `ALIGNED_NEWS_API_KEY` — Optional, for Pro tier

All credentials are accessed via environment variables only. No data is exfiltrated. The agent runs locally on your OpenClaw gateway.

## Permissions

- **Read:** X timeline, mentions, search results
- **Write:** Post replies, likes
